# 摘要

随着我国包装印刷产业的不断发展，包装印刷产品有着极其广阔的应用前景，而假冒伪劣产品通常用仿冒外包装来达到以次充好为目的。虽然以激光全息技术和防伪材料等新防伪方法也不断涌现，但印刷品复制显得过于复杂且对设备的依赖性比较大，增加防伪成本。因此，能够真正实现防伪识别简单，难于仿制以及控制成本是我们追求的目标。

本论文中，以基于莫尔效应(相位调整)的印刷防伪技术为理论基础，使基础科学的理论知识(物理光学)与印刷技术(半色调图像加网技术)相结合，将简单图像和文字等信息隐藏在预复制的平网色块或半色调图像中，进行了激光打印，真网点数码打样，机械打样以及胶印印刷等多种印刷实验。最后可以借助对应参数的光栅检测片与印刷图像在特定角度下密合，观察得到清晰的隐藏图文信息，并产生立体效果，最终实现去伪存真的防伪目的。

研究结果表明，该防伪信息可在数字印前流程中完成，在印刷中实现，可以应用在目前主流的印刷工艺，甚至激光打印中。该防伪印刷方法在不增加多余的生产成本的同时，完成对包装印刷品和保密文件的防伪保护，并且对应的光栅检测片参数唯一，使得其具有较高的保密安全性。另外，该技术的辅助工具光栅检测片价格低廉，识别方法简便易学，有利于在普通消费者和民众之间推广。

山此可见，该防伪技术具有非常广阔的应用前景，尤其是在产品的外包装上(如烟、酒、药品等包装的诸多领域)，既能准确地传达产品的市场形象，又为企业保护自身合法权益提供有力支持。

关键词：印刷防伪；光栅识别；莫尔效应；隐藏图文信息

# 1前言

# 1.1研究背景

防伪是一个古老的行当，我国早在3000 年前，商代就用了有效的防伪手段，封泥和符印就是其典型的代表，到了春秋战国、秦汉时期，这种技术更加盛行，比较典型的是调发军队的信物—虎符，之后出现了在瓷器、字画上添加印记或警示语来确定制作者身份、防止假冒的做法[。

而防伪作为一个行业的迅速发展则是在进入20 世纪之后，尤其是近些年，随着虚假票据、证明文件和假冒商品的泛滥，间接地刺激了防伪技术的进步。随着不法分子仿冒产品技术的提高，防伪压力日趋严重。当前假冒伪劣商品的交易额约占世界贸易总额的 $5 \% - 6 \%$ ，每年高达1500—1800亿美元，给品牌企业和名牌商品带来极大的破坏，使企业蒙受巨大的损失；在我国，假冒伪劣产品每年造成 2000 亿人民币以上的经济损失。各个企业纷纷加大在品牌创立方面的投资，在不断扩大企业和产品的知名度上不遗余力。但是在名牌产品创立的同时，各种各样的假冒伪劣产品也以惊人的速度和数量充斥市场。因此，我们必须加大防伪力度，严防假冒伪劣产品对企业和消费者的伤害。

在商品的包装方面，人们将大量高新技术和成果应用于包装产品中，在满足商品包装的防伪辨真的同时，并不失其艺术性。防伪技术已成为一个崭新的并在不断发展变化的专门技术，其一般可分为物理防伪技术、化学防伪油墨技术、生物防伪技术、材料防伪技术、计算机网络防伪技术、防伪印刷技术等。防伪印刷是众多产品防伪技术方法的核心技术之一。前，防伪印刷技术在我国是使用最普遍、识別最简单、成本最低的防伪手段[3,26]。

当今的包装印刷防伪为满足市场需求，其发展趋势从原材料(防伪承印物和防伪油墨)制造防伪，向设计制版防伪、信息化编码防伪、印制工艺防伪、个性化喷码防伪等方向转变[4,16,17.18]。我们平时接触的图像、文字大部分最终要通过印刷输出，如商标、产品包装、书刊、证书、证件、邮票、磁卡、出版物、影音制品封面、货币、有价票券、入场券、单据、广告、挂历台历、名片、护照等，这就使得防伪印刷技术显得异常重要。其中基于莫尔效应原理的光栅识别隐藏图文技术就是属于设计制版防伪类别，将基础科学的理论(物理光学)和知识与印刷技术相结合应用于防伪印刷技术中，本课题将重点研究该防伪技术的原理并对其进行开发与应用[67]。

# 1.2国内外研究现状

# 1.2.1国内研究现状

国内也有将光栅识别隐形图文防伪技术称作开锁式设计技术，但由于起步较晚，制作和应用的厂度与深度均有待加强。

国内已见诸市场的技术目前主要在产品包装和证件防伪上进行应用，以方便现场打假识别和认证真伪，采用扰视底纹安全设计技术，并使用解码片来识别包装物的真伪。如石家庄制药集团有限公司的“维宏"阿奇霉素片包装，神威药业的欧意品牌包装；联合利华的中华牙膏包装盒；美宝莲香皂包装盒等等。使用者可以通过特定频率的检测片在平网、实地色、渐变色、四色图像里面观察得到隐藏的文字和图像信息[8。

北京安信防伪技术发展有限公司自行开发的安全设计软件secuview正式投放市场并建立了安全防伪设计中心，以适应此技术在国内的旺盛需求，为国内客户获得独特的设计制作全程优质服务提供了便利。目前已将其应用于爱普生(北京)技术服务有限公司、美国AMD科技公司、中国长城电脑等的防伪标识、包装和服务证书中。[12,13,62]

目前，北京圣德安讯科技发展有限公司已成功开发出"解锁防伪技术”，此技术为制版领域的更高端技术，特别适合于包装领域的应用。其解锁防伪技术即是用一把特制的解锁工具看到在平网、实地色、渐变色、四色图像里面隐藏的文字以及图像信息[60]。

# 1.2.2国外研究现状

在国际上，最早开发成功并应用的隐形图文设计技术来自扰视图文(scrambledindicia)处理和相应的图像编码技术，此技术是美国图像安全系统公司(GSSC)和匈牙利的 jura JSP最早开发并付诸应用的技术[11,19,20]。

该技术是美国图像安全系统公司和匈牙利优拉设计公司共有的专利技术。[21,22]它是将防伪图案"打乱”、变形后叠加在钞票、邮票、包装印刷图案等印刷品上。人的肉眼看不见这种标记，目前的彩色复印机和数字扫描仪也无法复制这种防伪标记图文，只有在特制的解码镜下，防伪图文才能显现出来，并产生三维效果[23]。

瑞士洛桑的联邦理工学院的IAmidror和RDHersch等人研究了将莫尔效应的原理应用于防伪证明文件上。他们分别利用一维，二维和随机的莫尔强度分布，实现隐藏图像和文字在形状、位置、大小和方向上的变化，即隐藏图文并不固定，实现高度敏感不断变化的莫尔效应[24,25]。

隐形图文设计图文标记最早在安全产品中得到了广泛应用，以提高它们的安全性能。例如：钞票、身份证护照、邮票、税票和各种产品的防伪包装上。该技术经常与紫外荧光油墨和红外油墨一起使用[27,28,29]。隐形图文设计标记可以用紫外荧光油墨印刷，或者识别时不仅要用识别镜片，还要用紫外灯或红外灯，达到多重防伪的效果[4]。

目前，世界知名防伪技术开发公司在隐形图文识别技术方面进行了大量的改进和创新。美国GSSC公司创立了STEALTHSIR品牌，使其产品做到简单、灵活和高效。简单即无需添加设备，不会干扰原有工作流程和低培训要求；灵活即产品与防伪技术无接缝的整合，多样的应用和多种编码选择；高效即可靠的证明力，可作为法庭证据和被精英人士所认可[30,31]。

# 1.3防伪印刷的发展趋势

纵观印刷防伪市场，其发展趋势从原材料(防伪承印物和防伪油墨)印制防伪，向

设计制版防伪、信息化编码防伪、印制工艺防伪、数字水印防伪等方向转变。

# 1.3.1材料防伪技术

材料防伪是指采用特殊的材料达到防伪目的的防伪技术。防伪材料的种类很多，它与产品生产的工艺过程，有很大相关性。在印刷防伪中，一般把由于在印刷品中采用并且带来了视觉上的改变的材料叫做防伪印制材料[64]。

基于这样的分类，当前主要的防伪印制材料可分为防伪承印物和防伪油墨，而在承印物又可以细分为防伪纸张和防伪薄膜。

# 1.3.1.1承印物防伪

(1)防伪纸张一般均采用坚韧、光洁、挺括、耐磨的专用纸。造纸原料以长纤维物质为主。造纸过程中还要加入水印、荧光纤维、安全线等技术[61]。

水印技术：水印是在造纸过程中形成，融合了设计、雕模、制网、抄纸等复杂的工艺过程。图文部分的透光度的不同，在平常情况下不容易被看到，在透过强光观察时，才可显出预先设计的水印图文。水印纸最好使用圆网机生产。圆网机运转速度比长网机慢，水印不是压制生成的，而是通过纤维堆积在纸张表面形成，可以说水印与纸张一同形成，是模制处理过程。比长网机清晰，厚度可达40微米，比纸张本身还要厚些。水印纸早期主要用于钞票等有价证券的制造，如今大多数国家的钞票、重要的国际文件、设施图纸和各种证件也都用水印纸印刷。随着水印纸造纸技术的发展，商品包装应用水印纸印刷已经开始应用，并且将成为今后发展的重要趋势之一。

纤维丝技术：造纸时直接在纸浆中加入有色或无色荧光植物纤维，或者在纸张未定型前将纤维丝撒在纸张表面，可以制成具有防伪功能的纸张。有色纤维人眼即可在纸面看到，无色纤维在紫外线照射下方可显现，其颜色有红、蓝、橘红等，形念可粗可细，可长可短，依设计而定。纤维一般都是随机分布，故其疏密、嵌露多少各异。除了植物纤维外，近儿年也有将金属箔、塑料圆形小片掺入纸浆防伪的。

痕量添加物加密：利用生物具有抗原、抗体特异反应的原理，将极微量的抗原加入纸浆或纸的某一部分。检测时用相应的特异性抗体与之结合，通过显色、荧光等标记物反应的有无，辨别真伪，也可以在纸张中加入痕量化学元素分辨真假。有种球赛门票就是用这种纸印制，检查时只要拿浸有特制试剂的笔，在表面上划·下，划过处即显示为黑色[15]。

(2)在印刷防伪上，防伪薄膜一般最后要与印刷有图案的纸张复合为一体，故要求膜及上而的图案是透明的，不能遮盖底下的图案。不仅进一步提高防伪强度，同时还具有耐摩擦、耐溶剂的保护作用。这里简要介绍几种常用的防伪薄膜。

激光全息加无色荧光防伪膜：是把荧光防伪技术与普通激光全息技术有机地结合在一起，出离形层、有机薄膜基层、高光反射层、全息信息层、荧光图案层、黏合剂层等组成。其特点是荧光层不在薄膜的表面，而是在夹层中，这种防伪薄膜的防伪力度更大，不容易被仿制，而且耐摩擦、耐溶剂。激光全息加荧光防伪包装薄膜包括两种形式，一种是电化铝膜，另一种是薄(软)膜。后者是根据用户需要印制了无色荧光防伪文字或图案的透明膜，在自然光下看不出痕迹，但在紫外灯下能看到清晰的荧光防伪文字或图案[38]。

光学干涉变色薄膜：是建立在平行平版多光束干涉原理基础上，根据多层光学薄膜之间具有干涉效应的原理而设计的。其原理是：每个多层薄膜结构都具有其特定的反射光谱曲线，这个反射光谱随入射角的改变而发生变化，即随着观看的视角不同，出现颜色变化，例如呈现金黄、绿、褐、蓝、灰等颜色。光学干涉变色薄膜可以根据不同厂家的需要进行膜系设计，选配和改变膜层材料和膜层厚度，以达到预定的反射光谱及光学性能指标，实现对物体表面反射光谱及光学性能进行控制和选择的愿望，从而实现防伪的目的[14]。

揭显镂空膜：是一种松紧受隐藏图文调制的图层结构，在揭启出该膜形成的防伪标识时，显现阴阳相对、内容相同的两图文。这种阴阳相对的图文的变形率不相同，因此一旦揭启产生变形，阴阳图文就无法再完全重合在一起，从而可以保证该防伪标识不会被仿冒者重新利用。该膜有镀铝和不镀铝，定位和不定位之分。

核微孔薄膜：是利用重离子加速器或反应堆的辐射功能将透明的聚酯膜击穿，在其上形成透气、透水的防伪标识微孔结构。重离子微孔防伪标识分上色透印型、滴水消失型、隐形微孔型和综合型。通过显微镜观察微孔的孔径、孔密度和孔型，并与特设参数对比而辨别真伪[1]。

# 1.3.1.2油墨防伪技术

所谓防伪油墨,是在油墨连结料中加入特殊性能的防伪材料,经特殊工艺加工而成的特种印刷油墨，具体实施主要以油墨印刷方式印在票证、产品商标和包装上。这类防伪技术的特点是实施简单、成本低、隐蔽性好、色彩鲜艳、检验方便(甚至手温可改变颜色)、重现性强,是各国纸币、票证和商标的首选防伪技术[69]。

紫外荧光油墨：在紫外光 $( 2 0 0 \mathrm { n m } { - } 4 0 0 \mathrm { n m } )$ 照射下,能发出可见光 $( 4 0 0 \mathrm { n m } 8 0 0 \mathrm { n m } )$ 的特种油墨。根据油墨有无颜色分为有色荧光油墨和无色荧光油墨两种。现在主流的紫外荧光油墨的成分组成里，主要将化学合成的稀土有机络合物作为荧光材料常规的有机稀土荧光络合物具有制备简单,在油性介质中易分散、溶解、细化,在可见光下无色，紫外光激发下表现较强的荧光效果[68]。

日光激发变色防伪油墨：在太阳光下,能发出可见光 $( 4 0 0 \mathrm { n m } . 8 0 0 \mathrm { n m } )$ 的防伪印刷油墨。这种油墨从表面上看是由于太阳光作用而变色,实质上也是受紫外线照射而变色。目前我国市场上已开发应用的几种此类油墨在太阳光(也可在紫外光)下,即发生变色效果,可以从无色变紫、蓝、黄等色,也可设计为从有色到无色变化,是防伪材料中的新秀[70]。

磁性油墨：它是用带磁性的氧化铁粉混入油墨中制成的专用油墨,现阶段主要用于印刷银行票证的磁性编码文字和符号,具有记录和存储信息功效。将印有磁性编码的票证投人磁码识读器中可辨识真伪。将其适当使用在包装装潢上,可具有较强的较安全

的防伪性。

热致变色油墨：在加热作用下,能发生变色效果的油墨。根据变色所需温度的不同,可以分为手温型变色防伪油墨和高温变色防伪油墨。手温型变色防伪油墨是指在$3 4 ^ { \circ } \mathrm { C } - 3 6 ^ { \circ } \mathrm { C }$ 温度下,能发生变色效果的油墨。按照变色方式的差异,又可分为单变色可逆、多变色不可逆和多变色不可逆热敏防伪油墨。热致变色物质可分为无机盐类、配位化合物、有机物(染料分子)和液晶高分子等。其中在液晶油墨中加进了具有结品性能的化合物，液晶在微弱电流和温度的影响下,因晶格发生变化而显示触明暗图案与色彩。用这种墨印出的包装装潢品可随温度的变化发生反应。如果用于印刷日历,将会随着温度的变化而呈现不同的色彩,是一种很有生命力、有待开发的新型油墨材料[67]。

# 1.3.3数字印前防伪技术

一般数字印前防伪在传统数字印前的工作流程的基础上，为了满足产品防伪的特殊需要而会加入更多的处理步骤，如为了进行版纹防伪则需在设计产品包装外观时需要采用其它方法获得一定的图形，如还需要对产品添加条码并添加可变数据，从而达到预期的防伪效果。防伪技术在现代化的数字印前阶段可以实施的主要有以下几种防伪技术。

# 1.3.3.1版纹防伪

版纹防伪是利用一套专业的软件系统完全运用线条进行防伪设计，其特点是运用线条进行防伪,能产生丰富的线条及图案变化,防伪设计在制版中完成、在印刷中实现，不增加额外的制作成本,防伪性能高。其原理是：通过软件无法复制这种线条效果,通过防止扫描达到防伪目的。作为一种高精密度的防伪手段,版纹防伪以其多变性、灵活性开辟了广阔的创意空间,适用范围极为广泛,主要用于证件类防伪,包括有价证券防伪、票据防伪、证卡、护照、身份证等,如纸币上那些有线条组成的全部元素就是版纹。版纹的主要功能是安全防伪，但并不局限于钱币,随着社会的发展,票据、有价证券、证件等的印刷也运用了版纹,而且开始走向了更加广阔的包装市场。如烟酒包装防伪、化妆品包装、药品包装、食品包装、饮料包装等[59]。

版纹防伪是成本最低的、防伪效果极好的一种防伪印刷技术。完全运用线条进行防伪,能产生丰富的变化,而一般的通用软件根本做不出这种线条的效果,还可以根据光学、光谱技术、印刷技术等各种技术的原理,利用版纹防伪制作软件来实现诸如析光、潜影等各种防伪应用[6I]。

除了可以用线条进行防伪以外,还可将具有分形艺术特征图形图像用于产品的包装设计中。由于分形图片有这样的特征：无限自相似性及无限细致性，一旦把包装的图案确定,就很难利用原稿扫描复制的方法进行假冒,而且几乎是不可能从包装图案设计上增加防伪特征,能做到既易于识别,又无须增长印刷成本,况且只要生成原始分形图案的参数不泄秘,对于外界来说这个产品的包装就是永远唯一的,从而达到防伪目的。其次,可以把分形得到的图案加入到激光全息防伪图中,制作具有分形艺术特征的标志激光点阵分形全息图效果,出计算机产生分形图像可以叠加在客户的标识或商标图案上，甚至也可以作为标志其中的一部分，从而达到很好的防伪效果。

# 1.3.3.3加网防伪

随着数字印前处理技术的发展，印前处理的核心 光栅图像处理器RIP的功能得到了极大的加强，越来越多的RIP光栅图像处理器提供了对设计好的版面进行混合加网和局部加网的处理能力，其为印刷防伪的手段增加了又一利器[5]。在印前处理的最后阶段，也就是对图像图形文字构成的版面信息进行光栅化的时候，可以在RIP端有选择的对版面上的元素进行加网方式上的区别，产生了几种特种加网方式。

(1)无网点加网。主要用于钞票等有价票证的印刷，由于网点印刷应用广泛，技术含量相对小，较易仿造。为提高防伪效果，钞票基本上采用这种无网点加网印刷，并用多种印刷方式如凸版印刷、凹版印刷等组合印刷。

(2)新半色调加网。Adobe公司开发出了将两种网点图像交错配置于同一平面上的图像处理方法，即将大小两种网屏交错配置，在一个平面上形成两种图像，根据具有两种网点的细微结构的特殊半色调加网技术达到均等配置，这种情况构成第一图像和第二图像(埋入图像或潜影)的各个成分配置成为大的第一图像领域和小的第二图像领域。通常条件下，只能看清第一图像，在某种特定条件下，第一图像消失，第二图像的潜影浮现出来，这就是新半色调加网技术。此外，一般印刷所用的CMYK中的黑(K)墨，是以碳黑为主体的黑颜料，呈现出从紫外线区到红外线区的全区域的吸收。但将CMY重叠后总是利用其不吸收红外线的情况，当使用红外线照相机等特殊鉴定装置来观察时，便可看清用含有碳黑的黑墨印刷出来的图像。利用这种现象，将含有碳黑的条码信息(真信息)和不含有碳黑的同一色系的条码信息(假信息)配置在同一平面上，用红外线等的特殊装置仅可读取到真信息。目前，新半色调加网技术主要应用于防伪印刷领域。目前条码易被仿制的现象得到了很大改善[51]。

(3)艺术加网。艺术加网技术的基本原理是用各种形状的图形或图案代替常规网点，达到再现图像阶调和层次的目的。其内容和应用将在下一节详细介绍。

# 1.4.1数字加网算法

在普通的图像印刷中，都是用加网得到的网目调图像来模拟连续调图像，通过加网来表现图像的层次、阶调等。所谓加网，就是将连续调图像上不同的阶调层次分解成一个个不同密度的像素，每个像素以其着墨面积占该像素总面积的百分比的墨点来表达层次阶调的浓淡。主要发展演变历程是从照相加网，到电子加网，再到现在所广泛使用数字加网技术。电子计算机的普及应用使印刷向数字化方向发展，数字加网技术也逐步取代以往的加网而成为加网技术的主流[5。数字加网是指桌面出版系统中采用基于PostScript框架下的网点技术。根据记录点的分布状态，数字加网主要可分为调幅和调频两大技术。

# 1.4.1.1主流电子加网技术与应用

一个世纪以来,印刷业主要的都是采用半色调加网来实现连续调的图像效果。这种加网就是将原稿图像部分以网格的形式分成一个个网点,通过改变网格内网点的大小来再现阴影、彩色等效果。这种半色调加网我们称之调幅(AmplitudeModulate 简称AM)加网是一种周期性的加网方式。在印制图像时，通常利用到半色调(加网)技术，即利用不连续的网点来实现印品上层次以及浓淡变化。图像的阶调和层次靠网点来还原，网点是使原稿阶调再现的必要条件。常用的网点技术通常分为调幅加网(AM)和调频加网(FM)为主要形式，并包括一些基于前两种加网方法而变形出的其他加网方法。

[9]

(1)调幅加网是以网点的大小来表示图像的色调效果网点在空间出现的频率和单位面积上网点的数恒定不变通过改变网点面积来表现图像色调。调幅加网利用每个小方格中的网点面积的变化来表现原稿层次,即网点间距离相同而网点大小不同如图所示。从原理上讲着墨区每个网点上的油墨膜厚度是一定的依靠改变分色版上网点大小面积率来表现图像的阶调与色彩。调幅加网技术的特点是网点间隔固定决定于加网线数着墨点集中位于网格中心其面积大小是阶调信息的函数。调幅加网作为现在使用最普遍的加网方式之一，其优点是中间调再现良好层次过渡平滑自然缺点是受印刷条件限制一般加网线数限制在线以下,图像不够细腻采用高网线数时高光和暗调部分再现不理想同时存在着细节丢失加网产生龟纹、玫瑰斑、视觉上的撞网等缺陷因此既难以实现高加网线数印刷也难以适合四色以上的印刷。常见网点可以有不同的形状：正方形、钻石形、圆形、椭圆形、双点式等。

1.正方形网点正方形网点在 $5 0 \%$ 的面积率处扩张系数是最高的。

2.钻石形(菱形)网点在网点面积为 $2 5 \%$ 处菱形的长轴交接，产生第-一次阶调跳跃，在网点面积为 $7 5 \%$ 处菱形的短轴发生第二次交接，产生第二次跳跃，由于菱形网点交接时仅在两个顶角发生，其阶调跳跃比方形网点要缓和得多，因此用菱形网点来复制图像时印刷阶调曲线较为平缓，中间调范围内表现特别良好，故菱形网点适合于以中间调为主的原稿。

3.圆形网点同面积的网点中，圆形网点的周长是最短的，当采用圆形网点加网印刷时，图像中的高光和中间调处网点不相接，但在 $7 0 \%$ 处的暗调才发生网点相接现象，并且扩张系数也比较高，故圆形网点在暗调层次处的表现能力较差。很多印刷厂往往避免使用圆形网点。圆形网点一般适合于复制亮调较多，暗调较少的原稿。椭圆形网点与菱形网点类式，网点的交接也发生在 $2 5 \%$ 和 $7 5 \%$ 处。

同一面积的网点由于其形状不同，所以其周长也不一样。因此网点扩大也不相同，在常规网点形状方形、圆形、菱形中，同一面积下圆形网点面积率最大，所造成的网点面积扩大比率也就最大。以 $50 \%$ 着墨率情况下，方形网点成棋盘状。它的颗粒比较锐利，对于层次的表现能力很强。适合线条、图形和一些硬调图像的表现。圆形网点无论是在亮调还是在中间调的情况下，网点之间都是独立的，只有暗调的情况下才有部分相连。所以对于的采层次的表现能力不佳，四色印刷中比较少采用。菱形网点综合了方形网点的硬调和閲形网点的柔调特性，色彩过渡自然，适合一般图像、照片的表现。

(2)调频(FrequencyModulated简称FM)加网是非周期性的加网方式,是爱克发与连诺一海尔两家公司在年提出的一项高保真彩色复制新技术。在这种方法中，网点大小是相同的，但位置是随机变化的如图所示颜色和密度山给定区域中网点的数量密度控制。在调频加网技术中，一个半色调单元中多个点群的集合构成了形式上很特殊的网点由于这些点出现的位置是随机的所以这种网点没有固定的形状两个数值上完全相等的像素在采用调频加网技术形成网点时虽然记录点的数相同但它们出现的位置却不同。调频网点的面积固定不变靠改变网点在空间的分布密度密集程度来表现图像色调。同时它利用网点的大小来区分其精度网点越小越能复制出更丰富的层次和色彩[10]。

(3)组合(混合)加网。即微聚集网点技术加网。组合加网是融合调幅和调频加网的优点以充分利用CTP技术的高质量、高生产能力，针对高精度印刷，并在印前或印刷中提供一种能够毫不费力地完美再现原稿的加网技术。组合加网在高光区域和暗调区域它像调幅加网一样，通过大小相同细网点的疏密程度来表现画面层次变化。网点位置随机分布，并经过特别计算处理以使网点不相互重叠也不会间距过大。为了适应印刷还加了另外一种计算方法，即利用多个细小点子组成一个较大的印刷点。在中间调区域，网点的位置具有随机性，同时还可以像调幅加网一样对网点的大小进行改变。所以中间部分的网点兼具调幅网点的分布特性和调频网点的阶调表现方法。

(4)超频加网。它是在亮调和暗调区利用调频网点捕捉细节，在中间调区利用调幅网点完成平稳的渐变，以实现网点转换处的平滑过渡，兼以考虑到印刷机的印刷限度。通过精确计算找出图像，而不再受限于调幅网点与调频网点的切换处。

(5)高保真加网。出于自然界存在的大量颜色，用现有的纸张、油墨和4种油墨调和尚不能完全再现天然色彩原稿的色域，因此要采用超过四色来制作印刷品，其目的之一是为了扩大色彩表示范围，以更逼真地反映自然。高保真彩色就是采用超过四色进行分色印刷的印刷品。为了改善印刷品上某些颜色的饱和度，通过在C、M、Y、K4色基础上加印纯净、明亮的红、绿、蓝紫色，可使印刷图像产生更自然的反差和更新鲜的色彩感，这种印刷方式称为高保真彩色印刷[40]。

最初，调频加网主要在印刷高品质、精细印刷品方面应用，随着调频加网在印刷生产中的推广，发现调频加网可以有效增强低分辨率图像的印刷效果。其低分辨率图像的加网输出稿的细节再现效果可以与较高加网线数的调幅加网输出稿的印刷效果相媲美，所以印前排版可以使用较低分辨率的图像。因为没有加网角度，所以调频加网对套印不准不敏感。从而节省了印刷套准调节时间和调机准备时间，减少了调机的废品率，提高了印刷企业的生产效率[79]。

在加网过程中利用人眼在明视距离内只能分辨 $\mathbf { 0 . 0 1 m m }$ 大小以上的物体这一特征，如用各种形状的人眼不可见图形或图案代替常规网点，达到再现图像阶调和层次的目的，这样可以使图像在一定区域拥有人眼不可见的隐藏信息，从而达到防伪目的。普通加网技术常用的网点形状一般是圆形、椭圆形、菱形或方形等。方法可以通过调整网点函数的数学模型，灵活控制网点的形状，并用函数控制其之间相隔的距离，因此也就包括了几个重要的防伪特征。另外，可以利用网点扩大补偿过程中做出防伪作用，利用不同的函数，通过调频加网模式进行网点扩大补偿，从而做出防伪。总之，加网并不仅仅利用在图像真实再现的方面，还可以应用在防伪方面[80]。

# 1.4.1.2调幅加网算法

调幅加网技术是在加网网点数目不变的情况下，以改变网点的大小来表达图像层次的深浅。它以网格中心元素为基础，相邻两网点的中心距离不变，网点的排列遵循一定的规律，网点形状是人为设计的点型，用像素值的大小，控制着网点面积的大小。调幅加网技术中有有理正切加网技术、无理正切加网技术和超细胞结构加网技术。

调幅加网技术先后经历了有理正切加网、无理正切加网和超细胞结构加网技术。有理正切加网是数字加网技术的基础，而无理正切加网又是针对有理正切加网的不足提出的。顾名思义，当加网角度的正切值为有理数时，称这样的加网为有理正切加网，当正切值为无理数时，即为无理正切加网技术。后来，在这两种基本加网技术的基础上，又开发了新一代的数字加网技术即超细胞结构加网技术，这种新的加网技术有效地克服了有理正切和无理正切加网的缺点，将数字加网技术推向一个更高的水平。超细胞加网技术的思想：在有限的输出设备数值范围内，通过采用超大型细胞内设置了多个网点生成长点的方式，解决了精密逼近 $1 5 ^ { \circ }$ 和加网线数之间的矛盾[5]。

(1)有理正切加网(RationalTangent Screening)是数字加网的基础，它早于无理正切加网技术的出现。数字网目调网点与输出设备记录平面的记录栅格交叉取决于所选择的加网角度，在加网角度在为 $0 ^ { \circ }$ 和 $4 5 ^ { \circ }$ 时，每一个网目调单元的四个角点能够与记录栅格的角点准确重合。但在实践生产过程中，加网角度的选择却受到了限制。实践证明，当四色版的加网角度分别为 $0 ^ { \circ }$ 、 $1 5 ^ { \circ }$ 、 $4 5 ^ { \circ }$ 、 $7 5 ^ { \circ }$ 时，四色版因相互作用使得所产生图像出现的龟纹最小，但如果把数字加网技术产牛的网点角度设置在 $1 5 ^ { \circ }$ 时会出现一些问题。此时很可能网日调单元只有左下角与记录栅格的角点重合，而其它三个网目调单元的角点与记录栅格角点均不重合。之所以产生这一结果，是因为15度角的正切是一个无理数，它不能用两个整数之比表示。山于 $\scriptstyle \mathbf { a } = \mathbf { t g } - 1 ( 1 / 3 ) = 1 8 . 4 3 5 ^ { \circ }$ ，-般取它的近似值18.4度做为加网角度，因此，在Photoshop 加网中经常见到18.4度，这也是电分加网中最常采用的 $0 ^ { \circ }$ 、 $\pm 1 8 . 4 ^ { \circ }$ 和 $4 5 ^ { \circ }$ 加网组合的出来[63]。

有理正切加网是数字加网的基础，并且一度在印刷生产中得到了很好的应用。但是该种加网技术仍存在一些问题：实际生产中的加网角度 $1 5 ^ { \circ }$ ，但为了实现有理正切加网，将此角度改为 $1 8 . 4 ^ { \circ }$ ，出此产生了绝对误差为 $3 . 4 ^ { \circ }$ ；加网后实际得到的加网线数与指定的加网线数存在一定的偏差。除黄版外，MCK三色版的实际加网线数均会出现偏差，从而导致各色版的实际加网线数不同，使得图像的再现效果欠佳。

(2)无理正切加网(IrrationalTangent Screening)是伴随着计算机硬件的提升，为了解决有理加网存在问题，而应运而生的新的加网算法。为了解决无理正切加网网自调单元角点不能与输出设备记录栅格(像素)角点重合的问题，无理正切加网在实现时通常采取逐个修正法和强制对齐法，其中逐个修正法需要大量计算时间和储存空间，故通常选用强制对齐法。强制对齐法的工作原理为：取整无理正切角度a的对边dy 和邻边dx，强制网自调单元角点与记录设备的像素角点重合，使之形成有理正切网点。但是，强制角点对齐后衍生出来的问题是，实际得到的加网角度和加网线数与给定的值有偏离。

无理正切加网虽然解决了角度偏差问题，但是由于将网目调单元的角点略微移动，造成移动后得到的网目调单元的底边将比通常情况下的网目调单元底边略短，这意味着加网线数比通常情况下得到的加网线数略高。使得加网线数和加网角度构成了一个不可分割的统一体，具有相互制约的关系。理论上，加网角度可以是任意的，但无论是采用有理正切加网技术还是无理正切加网技术，加网角度总要受到限制。例如，当采用有理正切加网技术时，为了保证加网角度的正切值是有理数，必须使网目调单元的角点与记录栅格的角点重合，加网角度不能自由选择；对于无理正切加网而言，为了保证能正确地加网，必须采取措施强制网目调单元的角度与记录栅格角度重合，这同样限制了对加网角度的选择。因此，用PostScript技术加网时不能实现任意加网线数和任意加网角度的组合。

(3)超细胞结构加网(Super Cell Screening)是一种针对常规加网方式(有理正切加网)和无理正切加网的不足而开发的改进方案。超细胞结构加网技术的工艺意义在于，采用这一技术后，可以在输出设备有限的记录分辨力范围内通过采用超大型细胞并在每个细胞内设置多个网点生长点的方法，解决精密逼近15度角和记录分辨率间的矛盾，使得桌面制版技术能够达到甚至超过传统四色分色工艺所能获得的效果。

超细胞是一个由多个网目调单元组成的阵列，比如一个 $3 \times 3$ 的超细胞是由9个网目调单元组成的。出于一个超细胞是超尺寸的网目调单元(超大网目调单元)，超细胞网点的生长方式并不是像常规网目调单元那样从一个中心点开始，而是从多个中心点开始。举例来说，对一个山9个网目调单元组成的超细胞，它有9个中心点。只要超细胞的四个角点与输出设备的像素角点重合，则每一个这样的超细胞有相同的形状，并包含相同数量的网目调单元和曝光光点。

超细胞的尺寸与网点记录单元相比大得多，因此在输出设备的记录平面上有许多可以放置超细胞的点，使得超细胞的角点与记录设备的像素角点重合。因此，用超细胞结构可以以很高的精度逼近传统的加网角度，并使得各色版的加网线数大致相同，从而保证复制精度的提高。上述的内容就是 linotype-Hell 的高质量加网HQS(HighQuality Screening)、Adobe 公司精确加网AS(Accurate Screening)和 AGFA 公司平衡加网BS(Balanced Screening)技术的原理，也是所谓大瓷砖(Big Tile)或超细胞(Super Cell)概念的由来。

与前两种加网技术相比，采用超细胞结构能产生更接近于传统网点技术的加网角度和加网线数。一方面，加网角度以很高的精度逼近传统加网角度；另一方面，各色版的加网线数也很接近，四个色版可能获得的加网线数略高于给定的加网线数，但它们的数值十分接近，例如传统加网线数为133lpi，超细胞加网线数则为138.142lpi左右。各色版的加网线数相近对忠实还原原稿的颜色和层次变化，确保中性灰的实现是重要的保证。

用超细胞结构加网技术能提高加网角度的精度，但这样的提高时有代价的。比如，取出9个网目调单元组成的超细胞加网需要的运算时间比采用一个网目调单元进行有理正切加网需要的计算时间要多得多。实际上，数字加网时总要权衡：超细胞单元越大，可获得的精度越高，但需要更多的运算时间，且将降低系统性能。

超细胞结构在再现数字图像的灰度上工作得很好，可表现的动态范围大，可以用较低记录分辨率的激光照排机或者CTP系统生成网点，从而改善RIP光栅图像处理器和输出设备的生产能力，已经成为现代数字加网技术的共同基础。

艺术加网中网点的生成方法：传统点聚集型加网技术用的是有序抖动阈值矩阵。抖动阈值矩阵中数值大小和排列顺序决定了网点生成的大小和形状。可以用PostScript网点函数得到离散抖动阈值矩阵，然后将输入图像像素点的灰度值和矩阵阈值中的数值进行比较，得到位图图像的输出灰度级。艺术加网中不是用抖动方法生成网点，也很难用网点函数描述复杂的网点形状，其复杂的网点形状是用轮廓线的方法生成的。在轮廓线方法中，先是预先在网点定义空间中设计出每一个（关键灰度级(处的网点形状轮廓线。然后据此作内插运算（先线性插值获得控制点，然后用多项式样条得到其他曲线轮廓)，从而得到所有其他灰度级上的网点形状[39,77]。

网点排列形式：通过从网点定义空间（确定网点的形状)非线性变换到网点表现空间(确定网点的排列形式)，可以使网点空间排列方式富于变化，从而产生如膨胀、收缩和非线性变形等加网效果，这样得到的复制品无法被扫描(扫描会产生莫尔纹)，从而达到一定的防伪效果。

# 1.4.2莫尔效应

莫尔条纹作为一种光学现象，是由两个覆盖物相互作用所产生的(例如光栅与网屏之间)。“莫尔”一词来自法文，其原来的含义是波动的，或起波纹的。最早的时候法国丝绸工人发现当两块薄的丝绸织物叠在一起时，可以看到一种不规则的花纹。后来人们就把两组条纹叠加在一起产生的图形称为莫尔条纹。甚至在日常生活中，两块窗纱叠在一起时也可以看到这种莫尔条纹。莫尔条纹是指由单独结构体相互覆盖所产生的一个清晰的新图案，其中原始的结构体并不含有这个图案。随着科学技术的发展，莫尔现象理论广泛应用在工程设计和测量等领域。由于莫尔现象对细微位移、转动和形变极其敏感,因而可以在光学校准、结晶学和信息科学中的文档加密、防伪中加以应用[35]。

其实莫尔条纹的形成实质是光在通过光栅时衍射和干涉的结果。如图1-4所示，两个光栅片相叠加所产生的莫尔条纹(虚线部分)，其中Pi，P为光栅常数，两光栅的夹角为0。当P，P2，0取不同值时，会出现不同样式莫尔条纹。多数情况下我们主要应用横向莫尔条纹，即当P1=P2=P时，夹角0越小，产生的条纹越大，莫尔条纹周期越大；当夹角0足够小时，P有微小的变化，莫尔条纹周期则会有几何倍数的变化[36,73]

莫尔图案对微小位移和微小转动都非常敏感。只要互相重叠的两幅图案之间的相对位置有一点点的变动，都可能带来莫尔图案的十分剧烈的变化。比如最普通的直条纹的重叠，它们的莫尔条纹的周期满足公式：

$$
\mathrm { T _ { M } } = { \frac { \mathrm { T _ { 1 } } \mathrm { T _ { 2 } } } { { \sqrt { \mathrm { T _ { 1 } ^ { 2 } } + \mathrm { T _ { 2 } ^ { 2 } } - 2 \mathrm { T _ { 1 } } \mathrm { T _ { 2 } } \cos \alpha } } } }
$$

式中： $\mathrm { T } _ { 1 } , \ \mathrm { T } _ { 2 }$ 和a分别为两个直条纹图案的周期和它们之间的夹角。当原来两个直条纹图案的周期相等时，即 $\mathbf { T } _ { 1 } = \mathbf { T } _ { 2 } = \mathbf { T }$ 时，

$$
\mathrm { T } _ { \mathrm { M } } = { \frac { \mathrm { T } } { \sqrt { 2 { \bigl ( } 1 - \cos \alpha { \bigr ) } } } } = k \mathrm { T }
$$

式中放大率k与a有关，且a越小， $\mathrm { ~ k ~ }$ 值越大。如果a取非常小的值，那么原图案移动一个周期T的距离，莫尔图案也将随着移动它自身的一个周期 $\mathrm { { T _ { M } = \ k T } }$ 的距离。这样微小的移动使莫尔图案以k倍的速率移动[+2,75]。

# 1.4.3莫尔条纹及其防伪原理

运用莫尔现象是以当代的机加工技术为基础，可人为创造这种莫尔现象并加以利用。现代加工技术可以在金属或玻璃表面刻划出宽度相等、间隔相同的高精度线纹。仅在一毫米的间隔内即刻出200条线纹，而刻线精度可以达到在一米长度上仅有 $4 . 3 \sim$ 5微米的刻线误差。这种具有精密刻线的尺叫做光栅。用透明玻璃做的光栅叫透射光栅，用不锈钢做的光栅叫反射光栅。就透射光栅而言如果将两块刻线宽度相等、节距相同的光栅使其刻线平面平行且靠近，并使两块光栅的线纹间呈现一个微小的夹角，且在平行光束照射下就会在线纹的垂直方向看到明暗相间的条纹，这就是莫尔条纹[43,45]。

# 1.4.3.1印刷莫尔条纹

莫尔条纹是十八世纪法国研究人员莫尔先生首先发现的一种光学现象，从技术角度上讲，莫尔条纹是两条线或两个物体之间以恒定的角度和频率发生干涉的视觉结果，当人眼无法分辨这两条线或两个物体时，只能看到干涉的花纹，这种光学现象就是莫尔条纹。莫尔条纹对于半色调丝网印刷是一个潜在的问题。所谓半色调印刷，就是将连续调原稿通过照像或其他方法分解成大小不同的网点来表现层次的方法。暗调用印刷较大的网点来表现，亮调用印刷较小的网点来表现，同一色的网点之间，特别是多色印刷或四色印刷各色版网点之间会发生干涉形成莫尔条纹(如图1-6)[46,47,48]。

# 1.4.3.2莫尔条纹防伪种类及其原理

印刷防伪领域内，我们通常视印刷半色调网屏作为一个周期变化图案，把另一个相同周期的图案作为防伪检测片，将检测片覆盖于印刷品上，如果角度位置恰当的话，就会产生新的清晰图案。随着人们对莫尔效应防伪原理的不断研究，出现多个不同的防伪方式方法，主要分为以下几个类别：

(1)莫尔感应条纹，又称加网陷阱，通常将一系列的逐渐变换方向和频率的直线或者曲线组置于原始文件中。当仿造的文件通过标准的半色调彩色复制系统如电子扫描、胶印进行仿造时，这些花纹会产生一些强烈的莫尔条纹[49,50]。

(2)莫尔强度分布，该方法又可分为一维，二维和随机分布。本方法中所产生莫尔条纹不是唯一的，而是不断变化的条纹样式[53]。

(3)微透镜矩阵方法，这里用微透镜作为检测片使用。该微透镜利用视差效应让观察者观察到莫尔条纹的动态变化[58]。

(4)微型图案的莫尔放大，是利用莫尔条纹对细小量的放大作用。比如对原图案进行微小位移和角度的改变，产生相应的莫尔条纹的相对非常剧烈的移动、生成、吸收和大小、长短、粗细的改变以及从一种形状向另一种形状的衍变。当产生莫尔条纹的两幅图具有完全相同的重复周期时，莫尔条纹对原图案的放大作用清晰可见。图1-8为两幅图案的显微结构和所产生莫尔条纹， (a)的图案是"1"字形的周期性重复结构，(b)的图案是由许多透光圆点周期性排列组成的点阵，称作网板。如果把这两张图案重叠，并转动其中一幅图，使它们保持一个微小的角度,就会有莫尔图案出现，如(c)所示.这些莫尔图案也是"1"字形的图案，它实际上是(a)的图案的放大[54]。

现今传统工艺越来越多地被数字化工作流程所代替，同时也出现了许多以不同文件格式作为数据流的工作流程。根据数据流格式的不同，工作流程可分为PS、PDF和1-bitTIFF工作流程。1-bitTIFF文件就是以TIFF6.0格式保存的位图点阵数据，即二值单色图像(如图1-9所示)，是指图像在照排机和激光照排机或者CTP制版机上输出之前的位图数据，是RIP后的文档格式，与前两者相比，其具有安全可靠，能够及时发现文件中的错误以及“一次RIP，多次输出"的特点，在数字化工作流程领域重要的应用[65]。

推广1bitTiff图像的目的是防止图像数据在不同地点进行图像处理和输出时，可能出现的出于RIP加网的可变性(网点形状、角度的改变)以及不同厂家RIP特性的微妙差别，而导致的使最终图像偏离设计要求的情况发生。虽然TIFF/IT和PDF等标准可以防止文字乱码和图像替换的错误，但是对图像加网出现的问题却无能为力。用1bitTIFF做输入和输出有许多优点，一是它的输出准确性很高，而PDF要做到保证输出准确尚需时日。而且，将1bitTIFF作为中间数据的工作流程可以无需进行组织上和运用上的整合，顺利享用山CTP化带来的好处；二是，对其可以进行实用性高的高压缩[66]。

1-bitTIFF文件具有以下特点：

1．安全可靠作为RIP后数字化工作流程的数据流，1-bitTIFF文件是由RIP 生成，同时也记录了各种加网信息，包括加网角度、加网线数等，在前端形成的图形、图像、文字等信息都是以高分辨力的点阵信息固定下来并且不会再受后工序工艺的影响。因此，1-bitTIFF有"数码胶片"之称，以1-bitTIFF作为数据流的工作流程具有安全可靠的特点。

2．能够及时发现文件中的错误由于是先对PS或PDF文件进行RIP，再拼版，所以在RIP过程中就可以及时发现文件中存在的问题，从而保证了拼版文件的正确性。

3．最后一分钟修改在输出前的最后一刻，一旦发现某个版有需要改动的地方，对PS，PDF工作流程而言，修改完后需要重新进行整体地拼版；对1-bitTIFF工作流程而言，修改完后只要在拼版软件中将有错误的版替换掉即可，而不用全部重新拼版，从而可以极大地提高改错效率。

4.--次RIP，多次输出一次RIP生成的1-bitTIFF文件可以输出到多种设备上，如RIP后的数码打样，带1-bitTIFF输出服务端的照排输出设备，而计算机直接制版设备更是以1-bitTIFF文件作为惟-可接受的数据格式。

5．工作流程的自动化程度高从表面上看来，1-bitTIFF流程比RIP前的工作流程要复杂一些，但是在工作流程自动化的情况下，这种状况对操作人员而言是感觉不到的，低分辨力小档文件的产生和大版文件的输出都是在后台自动进行的，并不需要操作人员即时地干预。所以自动化程度高的工作流程能够很好地降低操作难度，提高生产的效率。

# 1.5研究目的、意义和研究内容

# 1.5.1研究目的、意义

虽然防伪技术种类繁多，涉及到诸多学科领域，其在印刷包装方面应用前景十分广阔，但如何真正实现防伪识别简单、难于仿制以及控制成本，将这点兼顾平衡是很难做到的，当然这也是我们努力和追求的日标[32,.3,34I。本课题研究的防伪技术性能

高，形式灵活适用范围广泛，制作成本相对较低，使用方法便捷，易于识别真假，抗扫描仿制等特点，该技术采取对产品防伪内容进行图形隐藏处理，利用光栅检测片显身隐藏图案，具有很高的隐蔽性和安全性。

该防伪技术的市场和产品的应用领域还需进一步细化，当前主要应用于有价票据，证件方面，应将其扩展至传统的包装印刷方面，日常文件的激光打印等领域，丰富其应用类别。

# 1.5.2主要研究内容

由于现在很多防伪手段成本过高，对于所要求的检测设备过于复杂，不利于其技术的广泛应用。因此，在结合当前包装印刷的环境，基础科学的知识和理论(物理光学)与印刷技术(半色调图像加网技术)相结合，在不增加额外的制作成本的前提下实现印刷防伪的最终效果。目的在于通过光栅物理光学特性与印刷半色调技术的结合可以获得合理可靠的防伪效果，从而加强其安全性，使之能够广泛应用在包装印刷，安全文件甚至日常激光打印等诸多领域。通过此课题的研究与开发，可提高商品和安全文件的安全保密性，对生产实践有着重要的指导意义。

本文主要研究的内容如下：

(1）对光栅基本结构及其光学特性分析，得出光栅对光线的折射规律以及对半色调图像的作用规律，继而与半色调图像加网技术相结合，得出对半色调图像网点设置时所需的算法。

(2)将隐藏图文加入平网色块进行模拟实验，采用激光打印机对平网色块进行输出，用对应参数的光栅片对印刷图像的防伪效果进行检验。在普通的激光打印中，在平网色块上实现了矢量图文的隐藏，得出基本的隐藏算法以及设定参数。

(3）选择灰度原稿作为源图像，并通过RIP光栅图像处理器输出1BitTIFF图像，将隐藏图文加入灰度半色调图像进行模拟实验，采用激光打印机对灰度图像进行输出，用对应参数的光栅片对印刷图像的防伪效果进行检验，并反复修正隐藏算法。实现了激光打印在灰度图像上的图文隐藏实验，并完善了隐藏算法，基本确定RIP的输出参数。

(4）选择彩色原稿作为源图像，并通过RIP光栅图像处理器输出1BitTIFF图像，将隐藏图文加入彩色半色调图像进行模拟实验，采用激光打印机和数字打样机对图像进行输出，用对应参数的光栅片对印刷图像的防伪效果进行检验。其中采用真网点打样的方法实现了对传统胶印的模拟，得到理想的防伪效果。

(5)将隐藏图文加入彩色半色调图像进行模拟实验，采用激光照排机进行菲林软片的输出，并利用圆压平式机械打样机进行打样输出，用对应参数的光栅片对印刷图像的防伪效果进行检验。其中采用机械打样的方法实现对传统胶印的模拟，得到理想的防伪效果。

(6)将彩色半色调源图像加入双层隐藏信息，采用CTP（计算机直接制版）系统进行印版的输出，并利用对开单张纸胶印机进行印刷输出，用对应参数的光栅片对印刷图像的防伪效果进行检验，可以观察得到二维隐藏信息。真正实现在胶印中隐藏信息的加入，同时达到双重防伪效果。

RIP光栅图像处理器软件中（如图2-10)，色空间反映了输出设备对色彩及图像数据的处理能力，其实质是告诉RIP用户想怎样处理待输出的文件：是否需要RIP分色、想输出二值加网数据还是连续调数据等等。其中Gray(Halftone）即为单色加网模式，制版软件前端分色，无需RIP分色，生成二值点阵数据；CMYKSeparations(Halftone)—RIP分色加网模式，前端制版软件生成的为复合色文件，需要RIP光栅图像处理器对其进行分色加网，为每个色版生成一个二值点阵图像。

对于RIP光栅图像处理器而言，所有照排机及直接制版机(CTP)都是单色半色调(需要加网)设备，因此当设备为照排机及直接制版机时，色空间只有以上两项可选。

其中：Gray(Halftone)通常也称为“前端分色模式”，要求排版软件前端分色，换言之，在排版软件中需要选择“分色”，生成的PS文件中已经包含了四个单色页面（假定只有CMYK 四色)，四个色版中的颜色数据都已确定，色版之间的压印(Overprint)、陷印(Trapping)等效果都已实现完毕，RIP光栅图像处理器对其只能进行单纯解释，已经不可能影响其颜色、压印、陷印等效果。CMYKSeparations(Halftone)通常也称为"RIP分色模式”。要求排版软件生成复合色PS文件，即PS文件中包含单个彩色页面，对于这类文件，RIP光栅图像处理器可以有很大的作用空间，可以对其进行色彩管理；对专色进行灵活处理；由RIP光栅图像处理器控制生成压印、陷印效果等等。Postscript3标准中的很多新增指令都要基于复合色PS文件完成，并且在印前作业流程中会发挥更大的作用[5]。

在RIP光栅图像处理器软件中，有七种不同加网算法，分别为Autobalance screen，HQSscreen，PostScript default，Enhance Hybrid Diffuse Dither，Hybrid Floyd-SteinbergDiffuseDither，Ordered DiffuseDither，FastDiffuse Dither，其中前三个为调幅加网方式，后四种为用于某些特殊要求的应用场合(如喷绘作业)才会使用的调频加网。在实验的RIP解释过程中，尝试了这三种调幅加网算法(Auto，HSQ，Postscript)，其中这三种加网算法对加网精度、网点玫瑰斑以及灰度层次的控制算法都各自不同。

# 2.3真网点数码打样实验

由于彩色激光打印输出精度达不到印刷品的精度，所以实验为了防伪达到更高精度，就需要采用高输出精度完成理想的防伪效果，实验选择数码打样作为打样实验的首选。所谓数码打样，就是把彩色桌面系统制作的页面(或印张)数据，不经过任何形式的模拟手段，直接经彩色打样机(多数为喷墨打印机)输出样张，以检查印前工序的图像页面质量，为印刷工序提供参照样张，并为用户提供可以签字付印的依据。

通常的数码打样系统主要包括两部分，色彩管理系统和输出设备。在色彩管理系统中包括两个核心技术，PostSCript解释技术和色彩管理技术。PostScript解释器保证用户的文件能得到正确的解释，并且与最终输出的胶片或版材完全一致，否则由于打样RIP光栅图像处理器和照排之间解释结果的不一致会产生差异；色彩管理技术能保证打样输出结果与印刷输出结果的颜色一致。

按照工艺流程划分数码打样可以分成RIP前打样和RIP后打样。在本实验所采用的真网点数码打样(如图2-15)，属于RIP后打样，一般是指数码打样RIP对最终输出

RIP后生成的1-BitTiff文件（即挂网后的1位Tiff文件）进行输出，在色彩管理的控制下，在打印机上得到与印刷品致的打印样张过程。其中1-BitTiff文件包含输出版面的全部信息，包括文字、版式、图像、图形及印刷网点结构（网点线数、网点形状与角度）的所有信息，所以据此输出的数码打样样张最忠实于最终印刷效果。因此实验选择真网点打样利用调频加网和相关算法模拟传统胶印的调幅加网[65,66]。

# 3结果与讨论

# 3.1加网线数与角度

不同RIP软件光栅图像处理器由于所采用默认算法不同，所造成的加网角度和线数也会出现差异。在实验中，激光打印和打样部分采用的HQ-RIP，而在胶印机印刷部分采用Esko-RIP，各自的加网角度和线数出现一定的差异，使得信息隐藏的算法和检测片的对应参数也出现了一定的变化。

国内常用的RIP 软件光栅图像处理器多为HarlequinRip(简称为HQ-RIP)，众多设备厂家已将HQ-RIP作为其输出平台的。在HQ-RIP软件操作过程中，可以利用了HPS功能，开启HQ-RIP的网点与网线补偿，从而修正实际的加网线数与网点成数同初始设置网线与网点成数的差别；Esko-RIP软件设置中，其加网角度由于算法的差异，和通常所说的角度互为补角，其加网类型默认为Eskoclassic,采用特有算法和优化技术，使得其实际加网线数与设置的线数有部分偏差，如175线的圆方形网点实际则为178.2lpi。

因此，在使用Esko进行RIP光栅图像处理器解释时，需要注意其加网角度和加网线数的变化，需要对应的改变信息隐藏的算法和检测片的对应参数，从而得到理想的防伪效果。

# 3.2加网算法

针对HQ-RIP光栅图像处理器来说，其调幅加网算法分为Auto balance screen，HQS screen,PostScript default 三类算法,其中最常见为 PostScript default,而 HQS screen被称为高质量高阶调加网技术，在本实验中，HQ-RIP的加网算法为软件默认 Autobalance screen。另外在 Esko-RIP 中，其调幅加网算法包括 Esko classic 算法。[5]

通过实验表明，Auto、Esko classic与Postscript的加网算法的图像与对应光栅片可以得到较为明显的莫尔效应，并可以观察到隐藏图像；HSQ加网算法的图像则效果不佳，并不能产生预想的莫尔条纹，也无法得到隐藏信息。主要山于算法在网点玫瑰斑控制算法的不同所造成的，在Auto、Esko classic 和Postscript三种算法中，加网角度与光栅检测片可以产生莫尔条纹，而HSQ算法所产生的加网角度和线数，则无法与光栅检测片发生莫尔效应。

因此在实际的应用和研究中，在HQ-RIP需采用Auto与Postscript这两种加网算法，一般情况下，加网类型法默认设置为 Auto balance screen 的调幅加网算法；在Esko-RIP中，加网类型可以默认设置为Eskoclassic调幅加网算法。

# 3.3输出分辨率

在源图像的输出打印或者印刷过程中，实验需要对输出分辨率特别留意，输出分辨率决定着隐藏信息能否成功隐藏和提取的成败。对于不同源图像，不同印刷方式和不同输出设备，其输出分辨率的设置也不尽相同。

(1)在QS图像和灰色色块的防伪中，其矢量转换为位图的过程中，分辨率设置为3000dpi-5000dpi所达到的效果更为理想，如果分辨率过低，会造成图像锯齿和模糊，而分辨率过高，会影响打印速度，并会使网点成数的大幅增加，进而影响防伪效果。

(2)在中低分辨率半色调图像的防伪中，由于输出设备为激光打印机，在RIP过程中，其理想的分辨率为 $6 0 0 ^ { * } 6 0 0 \mathrm { d p i }$ ，使得源图像达到理想的打印质量和防伪效果，如果设置过高的RIP输出分辨率(>720dpi)，而一般的激光打印机无法达到该分辨率，使得无法到达预期目的。

(3)利用真网点数码打样来打印高分辨率半色调图像时，RIP光栅图像处理器输出的1bitTIFF图像分辨率通常为2540dpi或者2400dpi，这是由于输出设备喷墨打样机的分辨率通常达到很高的级别，例如EPSON喷墨打印机输出分辨率最高可达2880dpi，HP喷墨打印机最高可达2400dpi，使得真网点数码打样可以模拟传统的胶印网点。

(4)在胶印印刷防伪中，结合胶印特点，RIP光栅图像处理器输出的1bitTIFF图像分辨率通常为2540dpi或者2400dpi，这是根据不同激光照排机或者CTP系统的输出分辨率所决定的。

# 3.4防伪技术的保密性

实验中所采用莫尔防伪技术具有独创性，方法和原理独特，将基础科学的理论(物理光学)和知识与印刷技术(半色调图像加网技术)相结合。

采用特种加网中的网线加网，不同于圆方网点或者方形网点等主流的加网手段，可以防止扫描仪的扫描，根据不同的源图像的特点，可以随时变化加网的网点形状和加入隐藏图像的色版，使仿制者无法确定，确保其安全保密。

扫描仪在扫描过程中，具有一定的局限性：不能识别彩色线条，只能识别单黑线条；不能识别有粗细变化的线条，只能识别均匀宽度的线条；不能同时识别线条和网点。基于以上三点，从而使扫描仪不能再现印刷品的线条形态。

所使用的光栅检测片参数要求与印刷品图文加网参数满足特定关系，而且还需在特定角度下才能出现莫尔条纹和隐藏信息，具有唯一性和保密性，极大地增加了不法伪造者的仿制难度和造假成本。

# 3.5图文的隐藏效果

实验中所采用莫尔防伪技术，利用相位的调整变化而产生莫尔效应从而显现隐藏信息，其添加的隐藏图文的隐藏效果是实验成果检验的一个重要指标，可以分成以下几个方面分别检查：

(1）加入隐藏信息的算法的改进，在真网点数码打样实验中，在2540dpi分辨率的条件下，加入隐藏图文时，改变加入的隐藏图文的相位参数，得出最佳相位参数，可以得到相对清晰的隐藏文字的同时，也可以做到极佳的隐藏效果，从而得到固定而准确的隐藏算法。

(2）印刷加网线数的选取，在传统胶印单色青打样实验中，在RIP加网过程中，实验选择100和175线分别进行对比组实验。其中线数100lpi样张对标识的隐藏效果并不好，已经影响到了原始图像细节再现；而线数175Ipi的样张对标识的隐藏效果较佳，与原始的图像相比基本无变化。故在实验中尽量使用较高的加网线数，从而达到良好的隐藏效果。

(3）线条网点与圆方形网点色彩再现的对比，在平版胶印机印刷实验中，A组、B组和D组所对应的相同图像比较，其印刷色彩再现程度差别并不大，将D组图像与对应参数的检测片结合观察，可以得到隐藏图文，并能观察到明显的莫尔条纹。由此可见，通过线条网点和圆方形网点的混合可以实现图文的优秀隐藏效果。

# 4结论

# 4.1结论

本论文中，以基于莫尔效应(相位调整)的印刷防伪技术为理论基础，使基础科学的理论知识(物理光学)与印刷技术(半色调图像加网技术)相结合，将简单图像和文字等信息隐藏在预复制的平网色块和半色调图像中，进行了激光打印，真网点数码打样，机械打样以及胶印印刷等多种印刷实验。最后可以借助对应参数的光栅检测片与印刷图像在特定角度下结合，观察得到清晰的隐藏图文信息，并产生立体效果，最终从而实现去伪存真的防伪自的。

本研究表明对其应用前景来说，该研究成果由于仅需要简单的光栅检测片，应用在商品包装上，易于被消费者识别和分辨，但检测片参数与印刷图文加网参数需满足特定关系，故使其保密性大大增强。

由于该技术应用在光栅解释后印前设计制版的数字印前阶段，使其几乎不会增加多余的生产成本，并不会改变原有的印刷工艺，在胶印、凹印、柔印等多种印刷方式上面都可以实现。另外其能应用在激光打印上，因此可以使用在日常工作生活上，成为简单的文件防伪方式，有利于人们的日常应用。也可以在源图像中加入不同的隐藏图文信息，对产品实现加密防伪，并可实现多层隐藏信息防伪，即将光栅检测片与源图像在不同角度下结合，得到不同的隐藏图文信息。除此之外，可以根据源图像产品的不同类别用途，根据其分辨率的不同，采用不同参数的光栅检测片，大大加强了其防伪的保密性。

# 4.2创新点

(1)将基础科学的理论知识(物理光学)与印刷技术(半色调图像加网技术)相结合，应用到包装印刷中。(2)在不增加额外的制作成本的前提下，使被包装的商品增加了防伪功能，并且产品具有防扫描和复印的功能。(3)该光栅防伪技术无需特种印刷油墨和特殊的印刷工艺，可应用在目前主流印刷工艺上，应用范围广泛。(4)该防伪技术的产品，绝大多数都是采用胶印等高分辨率印刷工艺，本次实验的成果不光可以应用在胶印等印刷工艺流程中，更重要的是还可以用在日常的激光打印中。对于一般民众来说，采用一般分辨率的激光打印机，便可以实现隐藏图文信息的加入，通过对应参数的光栅检测片，观察得到清晰的隐藏图文信息。

# 5展望

近几年来，随着社会和科技的进步，使得防伪技术种类和原理变得愈加繁多，涉及到诸多学科领域，其在印刷包装方面应用前景十分广泛和光明。本论文以基于莫尔效应(相位调整)的印刷防伪技术为理论基础，将简单图像和文字等信息隐藏在预复制的彩色图像中，进行了激光打印与传统印刷打样的模拟实验。实验结果表明：通过特定检测片在特定角度下，观察得到隐藏的图文信息并可产生三维效果，并可以应用在目前主流的印刷工艺，甚至激光打印中。该防伪印刷方法在不增加多余的生产成本的同时，完成对包装印刷品和保密文件的防伪保护。然而，本课题在总体上还存在一些不足，仍有很多方法需要进一步完善，主要有以下几个方面：

(1）加网算法的深入研究。由于时间和工作量关系，本次课题选择了已经很成熟的加网网形和RIP光栅图像处理器软件，并未将对原始的加网算法进行研究。建议在今后实验的开发中对加网算法进行研究，使得该防伪方式独立于RIP光栅图像处理器独立完成一部分防伪功能，提高其保密防伪性。

(2)完成光栅防伪软件的开发，以插件的形式将该软件与图像处理软件相融合，用于实际生产中。以插件的形式与当前主流的图像处理软件结合，更有利于该防伪技术的推广，适用于企业的大规模生产应用。

(3）将该防伪技术应用在企业的大规模生产。将防伪技术应用于包装产品中，改变企业的服务理念，将企业的发展道路引向高精尖的技术领域，通过知识进步，技术创新来发展包装印刷行业，增强企业市场竞争力。

(4)将该防伪技术与其他防伪技术结合使用，如与防伪油墨，防伪版纹相结合，最后实现多重防伪。

# 8致谢

转眼间就面临毕业了，时间的流逝之快让人无以察觉，内心思潮的涌动让人无以平静。在科大研究生生活的这段时间，让我难以忘怀，我衷心感谢每一位教导过我和关心过我的人。

首先我要感谢我的导师赵秀萍教授，赵老师工作非常繁忙，但是她仍旧关心着我们每一位学生，给我们以精心的指导。我的论文选题，资料的收集都是在她的指导、帮助下完成的，最重要的是她教会我学习、研究的方法，以及如何做一个真正的研究生。赵老师严肃的科学态度，严谨的治学精神，精益求精的工作作风，开朗豁达的人生态度深深地感染和激励着我，给我树立了一个学者和长者的伟大形象，使我有了奋斗的目标，让我终身受益。

同时我还要感谢唐万有老师，陈蕴智老师，蔡圣燕老师，陈永利老师，司占军老师，没有你们的教导，没有你们的付出，我也不会掌握到那么多的知识，没有这些老师的帮助我也不能如此顺利将这篇论文完成。

感谢我的父母和我亲爱的朋友们，你们总是在背后默默的支持我，在我遇到困难时给我鼓励与帮助。感谢我的同学们以及师妹师弟们，大家一起度过了美好的研究生生活，你们的意见让我的论文得以顺利进展。

最后，还要感谢天津科技大学以及包装与印刷工程学院。
