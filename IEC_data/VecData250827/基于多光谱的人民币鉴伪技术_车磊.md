# 基于多光谱的人民币鉴伪技术

车磊山东新北洋信息技术股份有限公司

摘要：随着科技的日新月异，各种伪劣假冒产品层出不穷，而最为危险的就是人民币假币的出现，其不仅会造成严重的经济损失，还会影响社会稳定，国家对于人民币鉴伪技术愈发重视，人民币诞生时，也诞生了一些鉴伪技术，但是近年来科技的发展，许多鉴伪技术已经不再适用于现今出现的假币中。目前，最为可靠的一项技术就是多光谱成像技术，利用假币在不同光谱下所呈现出来的与真币之间的图像进行真伪判断，利用该技术具有可靠度高、操作简单等一系列。现今假币犯罪猖獗，认识到多光谱下人民币鉴伪技术的特点对抵制假币，维护自身财产安全有着重要意义。

关键词：人民币；多光谱；鉴伪技术；特点；意义

# 一、引言

建国以来.我国的经济迅猛发展，各项科技也不断进步，我国的综合国力的依据大提升，但与此同时，因科技发展而导致的假冒伪劣产品事件也层出不穷，其中最为引人关注的就是假币事件。我国每年各地都有假币案件告破，假币面值小至几万元，多则上亿元。2009年6月北京警方破获贩卖假币案，面值约600余万元；2015年9月广东省公安厅破获建国以来最大的假币制造案件，假币多达2.1亿元；2016年1月江西省破获该省最大假币案，面值多达七千余万元等，一系列的假币案件让人们更加重视人民币真伪的鉴别，尤其是对人民币真伪鉴别技术的要求也越来越高。

现今有关人民币鉴别方法有很多种，在银行系统或者其他商业企业单位在进行人民币鉴别时候一般都采用验钞机进行，一般验钞机都综合了纸币的重量厚度以及纸币表面文字识别等技术，较为可靠。而在一般的用户手中没有验钞机就只能通过对纸币表面文字以及纸币颜色进行肉眼识别，并利用手指对纸币图案触摸来进行识别，这种方法在前十年可行，但是目前制造假币技术越来越先进，靠肉眼和触摸已经不能充分识别出假币。在目前市场上的验钞机鉴别人民币真伪一般分为两种即磁性监测和光学监测，这两种监测虽然目前看来较为可靠，但是还需要不断地改进，亟待新的鉴别技术的出现。故而认真分析现存的监测方法具有重要的现实意义。本文对基于多光谱的人民币鉴伪技术进行了仔细分析，为后期该技术的进一步发展与人们的普遍应用奠定一定基础。

# 二、基于多光谱的人民币鉴伪技术的主要内容

# (一)光谱图像的采集

在采用多光谱技术进行人民币鉴伪时，其首要的任务就是要进行纸币表面的光谱采集。因为纸币是通过不同的油墨印刷而成，纸币油墨在自然光线或者其他单色光下呈现出不同的颜色，通过相应的图像传感器对呈现出的颜色图像进行采集。多光谱技术中的图像采集大部分采用的是高速摄像头进行采集或者是利用接触式的传感器进行采集，高速摄像头进行采集时候虽然能够获得高清分辨率极高的图像，但是摄像头的精度要求特别高，相应的造价也很高，且其使用寿命较短，故而在多光谱纸币鉴伪技术中目前许多商业机构或者银行系统一般都用接触式传感器。另外，在进行图像采集时候除了对纸币表面的图像采集之外还包括对人民币上一些固定的防伪特征进行采集，之后再进行相应的处理。

# (二)光谱图像的处理

在进行图像采集之后，就要进行相关的光谱图像的处理，所谓的光谱图像处理就是将采集到的光谱图像进行分类处理以及特征的提取，将处于纸币表面上不同地方不同光谱的图像归类，并提取出一些相关特征，这样会有利于后期机器的识别。在光谱图像处理技术近年来已取得较大的成就，例如网格特征技术法、类哈如特征提取法、Fisher线性分析特征提取法等一系列方法，这些方法都能够很好地对采集到的图像进行合理地处理。而当前较常见的方法是网格特征技术法，该方法原理很简单，即以纸币的顶点和边线为划分依据，将纸币区域内划分为若干个不同的长方形或者正方形网格，并将每个网格的图像像素进行归纳总结。在归纳总结时候，除了要分析图像的像素之外，还要分析出每个网格的完整度以及与真币之间进行对比。不管采用什么技术进行光谱图像的处理，其最终目的就是为了区别其真假，如果图像处理少了与真币中对应出的对比，该技术就是失败的。

# (三)光谱图像的识别

图像的识别是鉴别人民币真伪的最后一步，这一步是定性人民币真假的关键性的一步，在进行图像识别的时候主要包括对尺寸的识别、特征向量间差异的识别、图像的灰度识别。图像的尺寸识别主要是为了判别人民币是否有损害，例如缺角、断裂等，常用的方法是网格特征技术法和类哈如法，特征向量间的差异识别主要是为了判断人民币的完整度以及为后期的灰度识别奠定基础，目前特征向量间的差异识别常常表现在分类器的使用上，最为常见的就是神经网络分类器，图像的灰度识别是确定人民币真伪最主要的一项，因为人民币图像在不同光线例如紫外、红外、自然光以及其他单色光或者混合光下的灰度均有巨大的差异，而通过对比集中不同光线下真币与假币之间的灰度均值、方差以及梯度等就可以较为准确地鉴别出所检查的人民币的真伪。

# 三、基于多光谱的人民币鉴伪技术中的注意事项

# (一)基于多光谱人民币鉴伪技术在新旧纸币鉴别中应用

随着时间的不断推移，为了保证纸质版人民币更好地流通，国家要定期地发行新版人民币并且回收旧版的人民币，但是新旧版人民币的交替是一个长期的过程。例如2015年发行的新版面值100元的人民币，如果要完全流通并且2005年发行的面值100元人民币全部回收至少需要5年的时间，而这段时间内如何利用多光谱人民币鉴伪技术进行新旧版的鉴别就成为了主要问题。所以在进行该技术研究时候，要注意利用光谱来进行鉴别，而不要去依靠纸币版面的亮度，只有通过不同的光谱的分辨，才能够消除因纸币版面亮度和模糊度带来的误鉴。除了采用光谱分析，还要利用色相的饱和度以及相关的系数折减来分析旧币，这样可以进一步保证旧币可以被很好地识别出来。

# (三)基于多光谱人民币鉴伪技术光色选择

目前人民币鉴伪技术中常用紫外线或者红外线进行，但是随着科技的发展，许多假币已非这两种光线照射能够识别得了，故而要采用新的光色或者改进紫外线以及红外线照射的技术。在笔者看来，采用多光谱人民币鉴伪技术时候最好能够充分利用自然光，可以通过对自然光照射下真假人民币呈现出来的图像的不同光谱进行灰度分析，从而辨别其真伪，这样才用该技术所设计的人民币验钞机或者识别机才能够被普及使用。其次，在进行光色选择时候，也要多选择几种光色，通过不同种光色照射下而鉴别出的纸币更加的真实可靠，并且这样可以在无形之中增加制造假币的成本，能够从根本上减少假币的产生。

# 四、结语

人民币是我国各项商业活动能够顺利开展的前提和基础，人民币流通的稳定性是我国社会稳定的重要保障，目前人民币已经开始在许多国家流通，人民币真伪鉴别技术在人们生活中越来越占据更加重要的地位。与一些发达国家相比，人民币鉴伪技术还较为落后，需要不断改进。所以，要加强多光谱的人民币鉴伪技术的研究，通过分析该技术中光谱图像的采集、处理以及识别等关键环节，找出其中存在的一些主要问题以及应注意事项，并提出相关解决办法，对我国的人民币真伪鉴别技术发展有着重要的现实意义，不仅可以保证我国社会、经济的稳定发展，还将为实现中华民族伟大复兴这一中国梦而奠定坚实的基础.
