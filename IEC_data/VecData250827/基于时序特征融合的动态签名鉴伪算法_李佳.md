# 基于时序特征融合的动态签名鉴伪算法

李 佳a，李庆武a，b，马云鹏a，丁惠洋a( 河海大学 a． 物联网工程学院; b． 常州市传感网与环境感知重点实验室，江苏 常州 213022)

摘 要: 针对现有签名鉴伪方法对高水平伪签名鉴伪准确率低的问题，提出一种基于时序特征融合的动态签名鉴伪算法。首先根据签名者落笔与提笔的时间节点建立动态时间轴，在签名过程中提取笔迹的压力和笔速两类时序特征; 然后在两类特征对应数据的基础上构建时序特征融合模型，通过一种多维空间模型相似性度量方法计算待测签名和样本签名的相似度，从而实现签名真伪性鉴别。实验结果表明，与现有算法相比，该方法进一步提高了签名鉴伪的准确率和通用性。

关键词: 签名鉴伪; 时序特征; 特征融合模型; 相似性度量

# Dynamic signature verification method based on timing feature fusion

Li Jiaa，Li Qingwua，b，Ma Yunpenga，Ding Huiyanga ( a． College of Internet of Things Engineering，b． Changzhou Key Laboratory of Sensor Networks ＆ Environment Sensing，Hohai University，Changzhou Jiangsu 213022，China)

Abstract: In order to increase the accuracy rate of signature verification for higher-level forgery，this paper developed a dynamic signature verification method based on timing feature fusion． First，during the writing process，it created a dynamic timeline，and extracted two timing features，the handwriting pressure and writing speed． Then it designed a fusion model of timing features based on the data of the features above． This paper calculated the similarity between the test signature and the sample signature by a similarity measure method of multidimensional spatial model． Finally，it realized signature verification by the analysis of similarity． The experiments show that the proposed algorithm has the advantages of high accuracy rate and good universality．

Key words: signature verification; timing feature; feature fusion model; similarity measure

# 0 引言

人体生物特征包括生理特征和行为特征，在生理特征方面，主要有 DNA、指纹、虹膜等; 在行为特征方面，主要有签名、语音、步态等［1，2］。与生理特征相比，行为特征具有更强的唯一性和可采集性。手写签名作为一种行为特征，已被广泛应用于金融、刑侦等领域的身份认证之中［3］。近年来，利用计算机进行签名真伪性鉴别一直是国内外学者研究的热点问题［4 ～6］。根据伪造水平的不同，可将伪签名分为简单伪签名和熟练伪签名两类［7］。简单伪签名即伪造者在仅知道伪造对象名字的条件下书写的签名，是一种伪造水平较低的签名; 而熟练伪签名作为一种高水平伪签名，是伪造者在掌握被伪造者签名笔迹的情况下刻意模仿产生的。后者的鉴伪难度远高于前者。根据所提取特征的类型，可将目前的签名鉴伪方法分为静态( 离线) 签名鉴伪［8 ～14］和动态( 在线) 签名鉴伪［15 ～18］两类。前者通过签名图像的静态特征分析笔迹的相似度，具有易获取、对设备要求低等优点。例如文献［9］提出一种基于旋转不变特征的签名鉴伪方法，根据签名图像中不同曲线的曲率及方向创建特征矢量，利用最长公共子序列( LCS) 算法鉴别签名图像的真伪性。由于特征单一，且在训练过程提取模板曲线的复杂度较高，该方法的通用性受到制约。文献［10］通过签名图像的局部梯度特征进行签名真伪性鉴别，利用 Harris 算法检测笔迹角点，将角点处的梯度方向作为特征，分析笔迹的相似度。因缺乏对签名全局信息的分析，所以该方法的有效性不足。

尽管静态签名鉴伪方法在能够有效鉴别简单伪造签名，但由于缺乏对签名过程性信息的分析，难以准确鉴别熟练伪造的签名。动态签名鉴伪方法提取签名的过程性特征分析签名的真伪性。尽管其对采集设备的要求较高，但能够弥补静态签名鉴伪方法对签名图像形状过于依赖的缺陷，鉴别准确率高。例如文献［16］采用一种基于速度特征的签名鉴伪方法，将提取的速度数据分为低速、中速和高速三个级别，选取对应中速部分的笔迹构建模板图像进行笔迹鉴伪。该方法避免了低速和高速数据的不稳定性对鉴伪效果的影响，但仅选取中速数据也使得处理结果缺乏稳定性。文献［17］提出一种基于特征水平分区的签名鉴伪方法。该方法将速度和压力两类动态特征各自分为高低两个分区，通过训练过程给不同分区赋予权重实现签名鉴伪。由于未考虑到不同采样时刻分区情况的差异，处理结果存在一定的局限性。文献［18］改进了文献［17］的方法，提出垂直分区的概念。该方法在签名的起始、中间和结束三个时刻，分析速度和压力两类特征的分区数据，通过训练过程给不同时刻的不同分区赋权重，取得了良好的鉴伪效果。但由于分区数目过少，随着伪造签名模仿水平的提高，该方法的性能明显下降。

随着互联网技术的发展与先进电子设备在金融、公共安全等领域的普及，动态签名具有更为广阔的应用前景。因此，本文专门针对动态签名鉴伪算法进行研究。由于现有的签名鉴伪方法在对高水平伪签名进行鉴伪时效果不佳，为此本文提出了一种基于时序特征融合的动态签名鉴伪算法。

# 2 时序特征提取

本文利用 Gaomon1060PRO 型数位板及其配套电磁笔提取签名的时序特征。该数位手绘板具有压力感应功能，笔迹会随电磁笔按压施力的大小而产生粗细变化，并能够实时记录粗细数据。因此，将该数位手绘板与绘图软件结合使用，可以在计算机端实现对签名过程的记录。时序特征提取主要分为签名样本预处理、压力特征提取和笔速特征提取三个步骤。预处理主要是对帧图像中的字迹部分进行灰度化、二值化以及骨架提取［19，20］处理，这里不再赘述，重点介绍两类特征的提取方法。

# 2. 1 压力特征的提取

签名是一种具有稳定性的行为特征。同一签名者在签名时，其字迹相同部位的按压施力大小也是稳定的。因此，利用压力 特 征 可 以 有 效 地 对 签 名 者 身 份 进 行 鉴 别。Gaomon1060PRO 型数位板能够以 $0 \sim 3 ~ \mathrm { m m }$ 宽度的笔迹反映压力大小。本文将完整笔迹视频按等时间间隔分为 $\mid m \mid$ 段，在每段中再以等时间采样的方式采集 $\mathbf { \Omega } _ { n }$ 帧，设笔迹总时长为 $L$ ，实时笔迹宽度 $d$ 与时间 $\mathbf { \chi } _ { t }$ 的对应函数如下:

$$
\begin{array} { r } { d = D ( \mathbf { \Omega } _ { t } ) \quad \mathbf { \Omega } _ { 0 \leqslant t \leqslant L } } \end{array}
$$

将 $m \times n$ 个时间点对应的笔迹宽度值用矩阵 $\pmb { K }$ 表示为

$$
\begin{array} { r } { \boldsymbol { K } = \left[ \begin{array} { c c c c } { d _ { 1 1 } } & { d _ { 1 2 } } & { \cdots } & { d _ { 1 n } } \\ { d _ { 2 1 } } & { d _ { 2 2 } } & { \cdots } & { d _ { 2 n } } \\ { \vdots } & { \vdots } & & { \vdots } \\ { d _ { m 1 } } & { d _ { m 2 } } & { \cdots } & { d _ { m n } } \end{array} \right] } \end{array}

其中: $d _ { i j } ( \ 1 \leqslant i \leqslant m \ , 1 \leqslant j \leqslant n )$ 为第 $\mathbf { \chi } _ { i }$ 段内第 $j$ 个时间点的笔迹宽度，其定义为

# 2. 2 笔速特征的提取

签名者的书写习惯具有稳定性。在两次签名的绝对笔速不同的情况下，按等时间间隔对书写过程进行分段，不同时段平均笔速之间的比值仍是近似相同的，因此本文提取笔速特征进行签名鉴伪。将完整笔迹视频按等时间间隔分为 $\mathbf { \nabla } _ { m }$ 段，在每段中再以等时间采样的方式采集 $n$ 帧，对每帧图像进行骨架提取，时间间隔为 $\Delta t$ 。

$$
\Delta t = \frac { L } { m \times n }
$$

将不同时间间隔内的骨架像素点数记为 $p _ { i j }$ $_ { _ { i j } } ( \ 1 \leqslant i \leqslant m \ , 1 \leqslant$ $j \leqslant n )$ ，将对应时间间隔内的平均笔速 $\boldsymbol { v } _ { i j }$ 定义为

$$
v _ { i j } = \frac { p _ { i j } } { \Delta t }
$$

将 $m \times n$ 个时间间隔内对应的平均笔速用矩阵 $\varrho$ 表示为

$$
\pmb { Q } = \left[ \begin{array} { c c c c } { v _ { 1 1 } } & { v _ { 1 2 } } & { \cdots } & { v _ { 1 n } } \\ { v _ { 2 1 } } & { v _ { 2 2 } } & { \cdots } & { v _ { 2 n } } \\ { \vdots } & { \vdots } & & { \vdots } \\ { v _ { m 1 } } & { v _ { m 2 } } & { \cdots } & { v _ { m n } } \end{array} \right]
$$

定义函数 $M = \operatorname { m i d } ( \boldsymbol { Q } )$ 表示将 $\varrho$ 中每一行元素数值按由大到小的顺序排列，取每行的中位数， $\boldsymbol { \mathcal { M } } = ( \ v _ { 1 } \ v _ { 2 } \ \cdots \ v _ { m } )$ ，归一化的平均笔速 $\ v ^ { \prime } { } _ { i } ( \ 1 \leqslant i \leqslant m )$ 定义如下:

$$
\boldsymbol { v ^ { \prime } } _ { i } = \frac { \boldsymbol { v } _ { i } } { \operatorname* { m a x } ( \boldsymbol { v } _ { 1 } \ , \boldsymbol { \cdots } \ , \boldsymbol { v } _ { m } ) }
$$

对签名中的每个字迹，按上述方式，采集笔速特征对应的数据，将此数据作为等间隔采样的五个时间点对应的笔速特征进行后续的模型构建。

# 3 本文模型

本文提出一种时序特征融合模型对压力及笔速两类时序特征进行融合。根据采集的特征数据，以压力、笔速和时间为三个维度，构建融合特征空间模型。分别以 $x , y$ 和 $z$ 轴表示归一化的笔速、时间和压力级。由于对签名中的每个文字按等时间间隔采样的方式取 5 帧图像，所以在 xoy 面和 yoz 面分别对应有五个坐标点，根据两类时序特征对应的数据拟合曲线，利用 xoy 和 yoz 面上拟合出的曲线确定 xoz 面上的对应曲线。最后，连接同一时刻三个面上的对应点，在空间坐标系第一象限内拟合出一个三维空间模型。图2( a) 和( b) 分别为某真签名以及伪签名中的“李”字对应的模型示意图。该空间模型能够反映签名压力和笔速的融合特征。其三条轮廓线分别表示压力随时间的变化、笔速随时间的变化以及压力与笔速两者间的对应关系。由于真实签名与伪造签名的模型有极大差异，而同一签名者多个真实签名的模型有较高的相似度，所以本文对签名时序特征融合模型进行相似性度量，以实现签名真伪性鉴别。

# 5 实验结果与分析

为了比较本文算法与现有的两类签名鉴伪算法［9，10，17，18］的优劣，验证本文算法的正确性和有效性，本文采集 100 人的4 组真实签名、3 组简单伪签名和3 组熟练伪签名，构建简单和熟练伪签名库。选取1 组真实签名数据作为比对数据，将剩余3 组真实签名数据与简单伪签名数据组合，构建包含 600 组签名信息的简单伪签名库，再将其与熟练伪签名数据组合，构建包含600 组签名信息的熟练伪签名库。

由于算法的安全目标是对不同场景、不同伪造水平的签名实现准确鉴别，本文在签名数据库的建立上着眼于样本的代表性。在签名场景方面，库中真伪签名的采集时间是每日的 $8 \sim$ 11 时、14 ～17 时以及19 ～21 时三个时段，采集地点包括实验室、教室和室外，三个地点的采集人数比例为 $2 \colon 2 \colon 1$ 。在伪造水平方面，仅使伪造者知晓伪造对象姓名，不提供具体笔迹，经过 $1 0 ~ \mathrm { m i n }$ 的稳定性练习产生简单伪签名。将伪造对象的签名笔迹提供给伪造者，伪造者在 $6 0 \ \mathrm { m i n }$ 的时间内进行模仿训练产生熟练伪签名。上述简单伪签名库和熟练伪签名库均包含签名的动态信息和静态信息，且签名样本具有代表性，因此本文在两类库上开展本文算法和现有的两类签名鉴伪算法的实验对比，验证算法性能。

# 6 结束语

针对现有签名鉴伪方法在处理高水平伪签名中的不足，本文提出了一种基于时序特征融合的动态签名鉴伪算法。首先结合动态时间轴，提取手写签名的压力和笔速两类时序特征;然后根据两类特征的对应数据，以压力、笔速和时间为基准，构建时序特征融合模型，通过一种新的相似性度量方法计算不同签名的相似度，从而实现签名鉴伪。实验结果表明，与现有签名鉴伪算法相比，本文算法具有更好的鉴伪效果，且因为摆脱了对字迹形状的依赖，在对熟练伪签名进行鉴伪时优势明显。因此，本文方法准确率高、通用性强。为降低算法复杂度，文中动态特征的采样频率有限，而适当提升采样频率可丰富模型的细节。所以，分析采样频率与鉴伪准确率的关系，进一步提升鉴伪效果，是后续研究的方向。
