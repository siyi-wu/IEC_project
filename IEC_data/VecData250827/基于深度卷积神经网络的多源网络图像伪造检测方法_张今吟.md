# 基于深度卷积神经网络的多源网络图像伪造检测方法$\blacktriangle$ 张今吟

# （国家工业信息安全发展研究中心  北京  100040）

摘要：针对现有检测方法在对多源网络图像伪造内容检测时，存在检测精度过低，无法准确判定伪造范围的问题。因此，本文章提出基于深度卷积神经网络多源网络图像伪造检测方法。利用深度卷积神经网络，获取多源网络图像，并对其进行融合。生成伪造图像数据集，实现图像伪造检测。通过对比实验证明，新的检测方法可以实现对军事图像伪造内容的准确检测，明确划分伪造内容范围，具备极高检测精度。

关键词：深度卷积神经网络；网络；伪造；图像；多源；检测

随着信息技术的迅猛发展，网络图像已成为信息传播的主要载体之一。然而，随着图像处理技术的日益成熟，网络图像伪造现象也越发严重，这不仅对公众的信息认知造成干扰，更对国家安全、军事保密等领域构成了严重威胁[1]。在军事领域，图像伪造可能导致情报失误、战略判断错误等严重后果。因此，研究多源网络图像伪造检测方法，对于维护信息安全、保障国家安全具有重要意义。军事领域对图像伪造检测技术的需求尤为迫切。传统的图像伪造检测方法往往局限于单一技术手段，难以应对复杂多变的伪造手段。本文旨在探讨深度卷积神经网络的原理、方法及其在军事领域的应用前景。通过深入研究构建一套高效、准确的多源网络图像伪造检测体系[2]。这不仅有助于提升我国在该领域的技术水平，更能为军事信息安全提供有力保障。

# 2  建立多源网络图像伪造特征检测模型

在针对多源网络图像伪造检测的任务中，引入ManTra-Net 这种广义图像伪造定位/检测的解决思路，可以为构建高效的检测模型提供有力的支持。ManTra-Net 作为一种先进的网络架构，其核心思想在于利用深度学习技术，特别是卷积神经网络（CNN），来精准地定位和检测图像中的伪造内容[4]。其借鉴了生成对抗网络（GAN）的对抗性策略[5]。在 GAN 中，生成器和鉴别器通过相互竞争，不断优化自己的性能。这种对抗性训练策略可以有效地提高特征提取的准确性和鲁棒性。在 ManTra-Net 中，类似的思想被用于提升模型对伪造内容的敏感度，使模型能够更好地识别出伪造区域。

图像操纵痕迹特征提取器是由三大网络模块构建的。初始阶段，一幅三通道的图像首先通过首个网络模块，称之为“First Block”。此块融合了三种不同类型的卷积层：Conv2D（带有 10 个 5x5 的卷积核）、BayarConv2D（带有3 个 5x5 的卷积核）以及 SRMConv2D（同样带有 3 个 5x5的卷积核）[6]。图像经过这三种卷积层的处理后，所提取的特征被有效融合，从而形成了丰富的特征表示。

紧接着，经过 First Block 处理的特征图被送入“MiddleBlock”。此网络模块继续对特征进行深入的卷积操作，以进一步提取图像中的高级语义信息。随后，这些高级特征再被传递到“LastBlock”[7]。作为最后一个网络块，LastBlock 进一步精炼和整合特征，最终生成一张高度抽象化的特征图。这张特征图不仅包含了图像的深层语义信息，还为后续的局部异常检测网络提供了强有力的输入支持。

# 3  生成伪造图像数据集与图像伪造检测

在对伪造图像和视频进行检测以及定位的任务中，数据集是不可或缺的一部分。通过分析大量的真实数据样本，可以更好地了解伪造技术的特点和规律，从而制定更为有效的检测策略[8]。因此，需要收集丰富的数据资源，包括真实的伪造图像和视频，以及与之对应的标签信息。通过对这些数据的处理和分析，可以建立更加准确、高效的检测模型，实现对伪造内容的精准识别和定位。生成伪造图像数据集是为了满足特定需求而创建的[9]。在生成伪造图像数据集时，可以利用 FGSM 生成一个对抗样本，在原始干净样本的基础上，添加一个人眼无法识别的微小扰动，使得添加该扰动后样本可以成功欺骗深度卷积神经网络分类器，以此达到成功攻击深度卷积神经网络伪造检测的目的。具体而言，FGSM 对抗样本生成方法的核心思想是利用损失函数对输入样本的梯度来构建对抗性扰动。首先，计算损失函数关于输入样本的梯度，随后提取梯度的符号信息，这表示在每个像素位置上扰动应增加或减少的方向。接下来，将这个梯度符号与一个小扰动值ε相乘，其中ε通常是一个很小的正数，如 0.01 或更小，以确保扰动的幅度在可接受的范围内。最终，将得到的扰动添加到原始样本上，生成对抗样本x_adν。当这个对抗样本被输入到深度卷积神经网络分类模型中时，由于加入的微小扰动，往往会出现分类错误。这种方法展示了神经网络对微小输入变化的敏感性，从而揭示了检测方法潜在的脆弱性。利用生成的伪造图像数据集，对上述检测模型进行训练，利用训练后的模型对多源网络图像伪造内容进行检测。

# 1  基于深度学习的人脸识别原理

1.1 深度学习理论

深度学习的核心思想是利用神经网络模型对大量数据进行学习和自我调整，从而实现对复杂问题的分析和解决。这种方法最初来源于人类神经系统的工作原理，基于“训练——预测——反馈”循环进行学习。

在深度学习中，神经网络通常由多层连接的神经元组成，每层神经元都对输入数据进行不同程度的抽象和处理，将低级别的特征转换成更高级别的抽象特征，最终输出对输入数据的预测。通过不断调整神经网络的权重和偏差，使其能够从数据中提取出最有效的特征，并实现更准确的预测和分类。

而卷积神经网络是在深度学习神经网络的基础之上，由输入层、卷积层、激活函数、池化层和全连接层构成的
