# 基于生成式对抗网络的多人脸图像局部伪造特征智能检测方法

朱振刚， 严海兵， 杨 萌（苏州科技大学 图书馆，江苏 苏州 215009）

摘 要： 多人脸图像相较于单人脸图像，其复杂性更高。 攻击者通常仅针对图像的局部区域进行篡改，加大了检测难度。 为此，本文提出了基于生成式对抗网络的多人脸图像局部伪造特征智能检测方法。 结合多人脸图像的模糊性分布特征，进行边缘识别检测与模糊信息分簇。 并建立超分辨识别模型，以获取面部阴影区域特征分值，实现多人脸图像特征分割。 在此基础上，利用生成式对抗网络生成逼真的虚假数据以欺骗区分器，结合径向基函数对支持向量机分类模型进行伪造检测。 研究表明，所提方法能精准检测出人脸图像局部伪造特征，适用于多角度人脸图像伪造检测。

人脸图像伪造检测技术是以真实人脸图像和伪造人脸图像的统计分布差异[1]、图像拼接引起的篡改痕迹和产生的图像的真实性等问题作为鉴定依据[2-3]。 人脸伪造可以在电影、电视等创作中得到运用，提升创作的观赏性[4-5]。 但是，人脸伪造技术的恶意应用[6]，不但会对用户的肖像权、名誉权造成严重的伤害，还会给政治、经济带来巨大的危害[7]。 因此，研究人脸图像伪造检测方法，具有很大的实际应用价值[8]。

文献[9]将非监督时序异常检测的思路引入到人脸伪造视频检测中，并将其转化为非监督的异常检测问题；再利用多粒度信息，如偏移、局部和时间等，从检测的视频中重建出一系列的人脸特征点；最后，得出原序列和重建序列的误差，通过峰值频率和重建误差的积分，实现了对人脸伪造视频的自动检测。 文献[10]设计基于胶囊检测法的伪造人脸检测方法。 利用局部 Xception 网络进行特征抽取，减少模型参数；在主体中，引入具有注意力机制的胶囊结构，实现对面部的精确定位；采用合成多维焦点为代价函数，提升人脸伪造的检测能力。 文献[11]提出了一种利用局部相关性表达进行多模态反差的方法，以实现对人脸伪造的有效检测。通过浅层风格表达的方法，度量层次之间的相关性来实现对图像的局部风格信息的编码，从而提高了图像在空域的判别性。 利用幅度与相位的互补特性，研究基于分块幅度与相位的双注意模型，在频率域内捕获局部相关性的不一致性，从而实现对人脸伪造的检测。 文献[12]提出一种新型的逆向残差混排方法对人脸进行识别。 通过逆向残差混排方法，降低网络中的参数，降低计算量；经过学习损耗函数，实现了在复杂背景及光照条件下人脸的检测与识别。 但是，随着伪造技术的不断发展，现有的检测算法难以实现对伪造数据的实时处理。 如何在获得较少的伪造数据的情况下，适应不断变化的伪造技术，不断地更新和优化检测模型，是当前迫切需要解决的关键问题。

生成式对抗网络（Generative Adversarial Networks，GAN）是一种强大的深度学习模型，能够生成高度逼真的图像。 通过训练生成式对抗网络，可以学习到图像中的真实特征和分布，从而用于检测伪造图像。 此外，生成式对抗网络的对抗性训练机制也使得其能够自适应地应对各种伪造手段，提高检测的准确性和鲁棒性。为此，本文提出了基于生成式对抗网络的多人脸图像局部伪造特征智能检测方法。 所提方法能够在较少计算量的同时实现人脸伪造特征点定位，提高了检测的通用性。

# 1 多人脸图像特征分割

人脸图像中，特别是多人脸图像，通常存在大量的不确定性和模糊性[13-14]。 这些不确定性和模糊性可能来源于光照变化、角度偏差、遮挡物以及面部表情等因素。 当只篡改如嘴部、眼睛等局部区域时，由于篡改面积比例较小，使得伪造特征具有较高的隐蔽性。 模糊粗糙集能够基于图像的局部特征进行分割，这对于检测局部伪造特征非常关键[15-16]。 模糊粗糙集是粗糙集理论与模糊集理论的结合，粗糙集理论基于集合中对象间的不可分辨性，能够有效地处理数据中的噪声、缺失和冲突，将不确定的数据转化为一组等价类。 模糊集理论则通过引入隶属度的概念，能够描述和处理模糊现象，即那些没有明确界限、难以精确分类的问题。 模糊粗糙集将两者相结合，既能处理不确定性，又能处理模糊性，从而更全面地刻画数据的本质特征，精细地分割图像，可以更加准确地定位到被篡改的区域，从而提高检测的准确率。

人脸图像局部特征模糊相关特征量可以表示为

$$
r ( 1 ) { = } 0 , v { = } 1 , a ( 1 ) { = } 0 , q { \in } Q
$$

式中， $r$ 为脸部图像的模糊相关特征量， $\mathbf { \sigma } _ { v }$ 为模糊相关系数， $a$ 是初始的面部图像特征矢量， $q$ 为目前脸部图像的像素值， $Q$ 为目前脸部图像的最大像素值。

通过显著性检测得到人脸图像特征的模糊集合的隶属函数[17]，明确每个像素或特征点属于某一模糊集合的程度。 迭代公式为

$$
F ( v + 1 ) { = } \frac { B U ( v ) j } { K [ K { + } U ( v ) ] }
$$

式中， $U ( v )$ 为人脸特征模糊集合的隶属函数， $B$ 为显著性指数 $, j$ 是面部图像的像素特征分布函数， $K$ 是分布系数。

利用基于多维信息的混合空间聚类算法[18]，优化图像检测并聚类图像边缘权值[19]，得出人脸图像局部特征

$$
Y { = } r ( 1 ) F ( v { + } 1 ) q
$$

模糊信息分簇结果可以表示为

$$
\Omega { = } Y { \frac { g ( x ) } { h ( s ) } }
$$

式中， $s$ 为图像的像素点， $g ( x )$ 为图像单帧的矢量融合函数， $x$ 是图像帧数， $h \left( s \right)$ 为代表图像信息的模糊集函数。

再通过模糊粗糙集[20]，对图像分割优化，从而提高检测的准确率。 此时图像的信息素矩阵为

$$
{ \cal W } { = } \Omega \left[ \begin{array} { c c c } { { \mathrm { \ c o s } \theta } } & { { - \mathrm { s i n } \theta } } & { { 0 } } \\ { { \mathrm { \ s i n } \theta } } & { { \mathrm { \ c o s } \theta } } & { { 0 } } \\ { { 0 } } & { { 0 } } & { { 1 } } \end{array} \right]
$$

式中， $\theta$ 为信息矩阵的欧拉角度。

在此基础上，建立人脸图像的超分辨识别模型，获得面部阴影区域的特征分值 $M$ ，完成多人脸图像特征分割，为后续的伪造检测提供更有价值的信息。 $M$ 的计算公式为

$$
M { = } W { \mathrm { m e d } } ( X _ { j - 1 } , \cdots , X _ { j } , \cdots , X _ { j + 1 } )
$$

式中，med（）为图像的超分函数， $X$ 是当前图像的最大帧数。

# 2 多人脸图像局部特征伪造检测

特征分割虽然能够突出关键特征，但也可能引入一些分割误差，导致某些特征被错误地划分或遗漏。 这些误差会影响到后续伪造检测的准确性。 多人脸图像中的伪造特征非常微妙和局部化，有时仅涉及到像素级的微小变化。同时，随着伪造技术的不断发展，新的伪造手段可能具有更高的隐蔽性和逼真度。基于模糊粗糙集的特征分割方法可能需要不断更新和优化，以适应这些新型伪造技术。 生成式对抗网络是一种无指导深层网络[21-22]，其网络结构由生成器和区分器组成。 GAN 能够自动学习并适应不同的伪造手段和技术。 通过训练 GAN，可以生成高度逼真的伪造图像，从而模拟各种可能的伪造情况。 同时，GAN 的区分器部分作为一个强大的特征提取器，可以从图像中提取与伪造相关的关键特征。 在此基础上，将 GAN 提取的特征输入到支持向量机（Support Vector Machine，SVM）中进行分类。SVM 作为一种经典的分类算法，能够处理高维特征，并且能够处理非线性和不平衡数据，实现对伪造特征的准确检测。 多人脸图像局部特征伪造检测流程如下：

步骤 1 确定人脸关键点的位置信息。

（1）将提取的多人脸特征图像 $I$ 通过第一层网络训练达到偏差估算 $S ^ { \prime }$ ，添加初始的图像关键点定位估算[23-24]，获得这一层的关键点位置 $S _ { 1 }$ 。

（2）计算用于将输入图像归一化至规范形状的仿射矩阵 $T$ ，获得校正的脸部图像 $T ( I )$ 以及图像关键点位置 $T ( S _ { 1 } )$ ，并且从上一个阶段中所生成的临界点位置估算值产生临界点热度图 $\textstyle H _ { t }$ ，得到公式

$$
H _ { t } { = } \frac { M } { 1 { + } \operatorname* { m i n } ^ { | | } ( g , f ) { - } s _ { i } | | }
$$

式中， $s _ { i }$ 是 $T ( S _ { 1 } )$ 中第 $i$ 个图像关键局部点定位， $( g , f )$ 是图像像素点。

经过多次的网络迭代训练，得到每个阶段的图像关键点位置，直至样本数量全部训练完毕。

采用关键局部点提取的方法，精确地提取出人脸的局部特征，并把图像的二维坐标点变换成多维度的特征矢量，得到人脸关键局部点的位置集合。

$$
{ \left[ \begin{array} { l } { g _ { 1 } , f _ { 1 } } \\ { g _ { 2 } , f _ { 2 } } \\ { \vdots } \\ { g _ { n } , f _ { n } } \end{array} \right] } \Rightarrow { \left[ \begin{array} { l l l l l } { g _ { 1 } } & { f _ { 1 } } & { \cdots } & { g _ { n } } & { f _ { n } } \end{array} \right] }
$$

式中， $n$ 为样本图像的二维坐标个数。

步骤 2 将人脸关键局部点位置集合和对应特征输入到生成式对抗网络中， 生成器估算原始图像特征分布，通过引入随机噪声提高模型的泛化能力，从而产生非常真实的虚假数据，以达到欺骗区分器的目的。区分器所要做的就是判断一个数据是否真实[25]， $b$ 为实际数据的输入参数，若 $b$ 为一幅图像，则输出代表此图像的真概率 $D$ 。 若为 1，那就是 $100 \%$ 的真图；若为 0，那就是假的图像。 这样， $G$ 与 $D$ 就形成了一种动态的博弈，以此达成一种动态的纳什平衡。 对抗训练的程序数学描述为

$$
V ( D , G ) = H _ { t } E _ { b p _ { b } } [ \lg D ( b ) ] + E _ { z p _ { z } } \{ \lg [ 1 - D ( G ( z ) ) ] \}
$$

式中， $V ( D , G )$ 是生成式对抗网络训练过程中的损耗函数， $G$ 为生成式对抗网络的产生器， $D$ 为生成式对抗网络的区分器，$z$ 为添加的随机噪声， $E$ 为数学期望函数， $p _ { b }$ 为真实数据的分布概率， $p _ { z }$ 为噪声分布概率。

步骤 3 由于伪造特征和真实特征之间存在复杂的非线性关系，使用径向基函数作为核函数可以帮助 SVM 更好地捕捉这些关系[26-27]。 径向基函数具有局部响应的特性，对于输入空间中距离较近的点有较大的响应， 而对于距离较远的点响应较小。这种特性使得径向基函数能够关注到局部细节，对于检测局部伪造特征非常有利。 因此，通过径向基函数 $\kappa ( g _ { n } , f _ { n } )$ 对支持向量机分类模型进行伪造检测

$$
\kappa ( g _ { n } , f _ { n } ) { = } \frac { e ^ { - \gamma \vert \vert g _ { n } , f _ { n } \vert \vert } } { V ( D , G ) } , \gamma { \geq } 0
$$

式中， $\gamma$ 为核函数参数。 使用公式（10）对参数进行优化，并使用交叉验证找出实际图像分类中精确度最高的参数值， 实现伪造检测。

# 4 结语

由于伪造图片具有很强的隐蔽性，使得人们很容易将其应用到一些不道德的、恶意的行为中，例如制造虚假信息、通信诈骗等，对人民、社会造成了极大的负面影响。 为此，本文提出了一种基于生成式对抗网络的多人脸图像局部伪造特征智能检测方法。 考虑人脸图像的局部像素特征，通过显著性检测得到模糊集合的隶属度函数；通过模糊粗糙集分割优化图像，提高人脸图像局部特征提取质量。 通过主分量分析法定位图像关键点，构建基于支持向量机模型的分类器，基于生成式对抗网络方法实现对伪造人脸与真实人脸的有效识别。 由实验可知，所提方法对人脸伪造特征点的定位准确度高，检测精度高，可以用于多种角度的人脸伪造图像的检测。 然而，该研究没有考虑到人脸图像可能在不同尺度、不同视角下呈现，对伪造特征的检测带来的挑战。 在未来的研究中，通过引入多尺度特征融合策略，使模型能够同时捕获不同尺度下的伪造特征。 同时，通过数据增强和迁移学习技术，提高模型在不同视角下的泛化能力。

# Intelligent detection method for local forged features in multi-facial images based on generative adversarial networks

ZHU Zhengang， YAN Haibing， YANG Meng（Library，SUST，Suzhou 215009，China）

Abstract： Multi-facial images are more complex than singlefacial images. Attackers usually only tamper with the local areas of the images，which increases the difficulty of detection. To this end，an intelligent detection method for local forged features of multi-facial images based on generative adversarial network is proposed. Referring to the fuzzy distribution characteristics of multi-facial images，we detected the edge recognition detection and clustered the fuzzy information. The super resolution recognition model was established to obtain the feature score of facial shadow area and realize the feature segmentation of multi-facial images. Based on this method，the generative adversarial network generated realistic spurious data to cheat the discriminator，and performed forgery detection on the SVM classification model in combination with radial basis functions. The experimental results show that the proposed method can accurately detect the local forged features of facial images，which is suitable for multi-angle face image forgery detection.

Key words： facial image forgery；generative adversarial network；fuzzy information method；edge perception；support vector machine

# Visual analysis of multivariate temporal data based on smooth spatiotemporal representation learning

WANG Yunzhe， CAO Tianyuan， TAN Wei（School of Electronics ＆ Information Engineering，SUST，Suzhou 215009，China）

Abstract： Multivariate time series data exhibit complexity in temporal and spatial dimensions，making their processing and analysis challenging. This includes analyzing the complex interactions among variables and capturing their patterns as they evolve over time. In addition，effectively visualizing and interpreting these patterns is also an important task. This paper proposes a dual -module Multivariate Spatio Temporal Encoder （MSTE） model. First，within a single time window，the static graph model is used to represent the relationships between variables. Second，a dynamic graph model is employed to encode the evolution of these relationships across consecutive time windows. This model not only learns the temporal embeddings of individual variables but also generates smooth spatiotemporal correlation embeddings for dynamic pattern extraction by considering the stable transitions in the relationships between variables. Furthermore，this paper designs and implements an interactivevisual analytic system for data exploration，enabling a multi-scale，multi-dimensional analysis of the correlations in multivariate time series data and their temporal evolution based on the features learned by the MSTE model.

Key words： multivariate correlation；spatiotemporal embedding；visual analysis
