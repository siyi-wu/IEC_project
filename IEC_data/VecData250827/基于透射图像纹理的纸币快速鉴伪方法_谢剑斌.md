# 1 引 韓言

纸币是国家发行的主要货币符号在经济活动中起着重要作用。某些不法分子利用假币、变造币严重扰乱社会经济和金融秩序损害人民群众利益。常用的伪钞鉴别技术是通过磁性检验和光学检验等手段对纸币的磁性安全线、磁性油墨、红外、紫外等防伪特征进行检测［1-3］。这些方法原理简单、容易实现、鉴伪性能稳定。但是随着造假技术的不断提高出现了专门针对某些防伪特征的伪币譬如变造币可以通过磁性安全线、红外特性、磁性油墨等检测；用粘贴安全线等手法制作的伪钞可以通过磁性安全线检测。现有的纸币防伪手段正面临着越来越严峻的挑战。

纸张防伪、油墨防伪和印刷防伪等技术在纸币防伪中得到广泛的应用各国均有自己独特且严格保密的纸张配方在印钞纸生产中还加入水印、安全线、彩色纤维和无色荧光纤维等防伪措施；在印制过程中使用难以仿制的防伪油墨采用先进的雕刻凹版制版技术和平版、凹版、凸版相结合的多次套印技术［4］造成真币在表面的粗糙程度上与假币有很大的差别：真钞纸张特殊线纹墨层厚实颜色厚重纸面上凸出的线纹有一定光泽用手触摸有明显的凹凸感；伪钞纸张疲软、手感平滑凹凸感较弱。由于检测和分析方面的困难这些防伪特征并未在纸币鉴伪中得到应用。文献 ［5］提出了一种利用光电传感器对纸币凹版印刷特征进行检测的纸币鉴伪方法并在一种新的专业级金融防伪设备中得到了很好的应用但是该方法检测的特征较为单一没有充分发挥纸币的综合防伪功能成本也较高。

本文针对真伪纸币光学透射图像在纹理粗糙度上的差异提出一种基于透射图像纹理的纸币快速鉴伪方法。该方法基于纸币的透射图像通过对图像行或列灰度轮廓的分析提取纹理粗糙度参数反映真伪纸币在纸张、油墨和印刷工艺等方面的差异。

纹理可以看作是由图像像素组成的具有一定形状和大小的集合是图像的灰度统计信息、空间分布信息和结构信息的综合反映是图像表面的固有特性。当光线照射到纸币表面时部分光线发生反射部分光线发生折射如图1所示。

在折射过程中物体的透射图像与物体本身的吸收系数、反射系数和厚度等特性有关。如果入射光线的光强为 $\mathbf { I } _ { 0 }$ 透过介质后的透射光强为I则I和I存在如下关系 ［6］：

$$
\mathrm { I } = \mathrm { I } _ { } ( 1 - \eta ) \mathrm { e } ^ { - \alpha \mathrm { d } }
$$

式中： $\alpha$ 为介质吸收系数 $\eta$ 为介质反射系数d为介质厚度。

对于纸币而言印钞纸张的质地和厚度油墨的成份、颜色和厚度印刷的方式和技术水平都会影响到 $\mid \alpha _ { \setminus }$ $\eta _ { \setminus }$ 、d的取值从而影响票面各部分的透射光强。由于伪钞所采用的印钞纸张、油墨和印刷工艺不可能完全和真币做到一致两者的透射图像纹理也必然会有一定的差别在某些特定区域这种差别相当明显。图2分别列出了第五套人民币第二版百元券正面真、伪币在国徽、隐形文字及毛主席头像头发等部位的透射图像纹理清晰地反映出这种差别。

粗糙性是人们区别纹理时所用的最基本特征之一在纹理分析中占有举足轻重的地位［7］。为了对纹理粗糙度进行有效的数学描述Rosenfeld提出了基于最佳尺寸的纹理粗糙度算法［8］Karu和Tamura等人所使用的纹理粗糙度也都是基于 Rosenfeld纹理粗糙度算法［9-10］。该算法是从水平或垂直方向上通过计算像素的多邻域边缘灰度变化求取纹理的平均宽度以此作为纹理粗糙度的评定参数其结果符合人眼的视觉感受；但是该算法主要计算纹理的平均宽度对纹理细节的分辨能力不强尤其是对于真钞和伪钞这种纹理相似性极高的图像不能有效地反映它们之间的差别。

在机械行业中表面粗糙度是指物体表面由较小间距和微小峰谷所组成的微观几何形状特征并有一套科学、合理、快速的特征评定参数［11-13］。本文在深入研究透射图像纹理的基础上参考表面粗糙度的描述方法选取适用于纸币真伪鉴别的纹理粗糙度评定参数。

本文所使用的纹理粗糙度特指图像中由像素灰度值所形成的灰度轮廓以及由其轮廓间距和峰谷所形成的几何形状特征。

定义1 提取图像某行 （或某列）灰度值按其像素序号画出该像素灰度值所形成的曲线称为图像行 （列 ）灰度轮廓。该行（列）的平均灰度值称之为该行 （列）的算术平均中线（meanline）用 M表示。由相邻的一个轮廓峰和轮廓谷所形成的曲线称为轮廓单元 （profilecel）如图3所示。

本文所采用的纹理粗糙度特征参数提取算法直接基于图像灰度值不需要设定初始数值只需进行一次逐点扫描即可求出所有灰度轮廓参数。如设所需处理图像像素总数为 $\mathbf { \tau } _ { \mathrm { m } } \times \mathbf { \tau } _ { \mathrm { n } }$ 纹理平均灰度轮廓宽度为W在按列计算各轮廓单元参数时约需要进行 $4 ~ \mathrm { m n }$ 次的加减法运算在计算纹理粗糙度的特征参数时仅需要进行 $\mathrm { m n ^ { + } }$ $( 2 5 \mathrm { m n / w } )$ 次的加减法运算共计需要进行 $5 \mathrm { { m n } + }$ $( 2 5 \mathrm { m n / w } )$ 次的加减法运算乘除法的运算量可以忽略不计。因此本文算法的计算量非常小完全满足纸币实时检测要求。

# 5 结　 论

由于严格的生产工艺和质量要求真钞在纹理粗糙度参数上呈现出较强的一致性各个特征参数具有较小的动态范围；同时由于在纸张与油墨的材质性能、印刷工艺和制版技术方面的不同真伪钞在光学透射性能上存在明显差异。因此在光线透射条件下本文通过提取纸币的纹理粗糙度参数进行纸币快速鉴伪；而且通过实验方法获取不同国家纸币的提取图像区域和纹理粗糙度参数可以形成多国纸币的特征数据库。该方法计算量小、速度快、精度高、使用方便可以用于纸币快速鉴伪设备和系统之中。
