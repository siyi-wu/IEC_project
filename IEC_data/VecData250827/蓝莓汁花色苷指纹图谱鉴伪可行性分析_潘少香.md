# 蓝莓汁花色苷指纹图谱鉴伪可行性分析

潘少香，闫新焕 \*，谭梦男，刘雪梅，孟晓萌，郑晓冬，李志成，曹宁（中华全国供销合作总社济南果品研究院，山东 济南250014）

摘 要：蓝莓浆果产量低、成本高，因此果汁掺假、造假等情况时有发生，亟需一种快捷、可靠的蓝莓果汁掺伪识别技术。采用液相色谱法检测蓝莓汁中的花色苷，优化后的色谱方法为 $\mathrm { A c c l a i m ^ { \mathrm { { T M } } } } 1 2 0 \mathrm { C } 1 8$ 柱（ $2 5 0 \ \mathrm { m m } { \times } 4 . 6 \ \mathrm { m m } \ 5 \ \mu \mathrm { m } )$ ；流动相分别为 $1 0 \%$ 甲酸水、水:甲酸:乙腈（4:1:5），采用梯度洗脱；柱温 $4 0 \ \%$ ；流速 $1 . 0 \ \mathrm { m L / m i n }$ ；检测波长 $5 3 0 ~ \mathrm { n m } _ { \odot }$ 。进而建立了蓝莓汁中花色苷指纹图谱快速检测技术，并比较了石榴汁、树莓汁与蓝莓汁花色苷指纹图谱的差异，讨论了花色苷指纹图谱法鉴伪的可行性。结果表明，建立的蓝莓汁花色苷指纹图谱技术能够用于蓝莓汁的鉴伪。

关键词：高效液相色谱法；蓝莓汁；花色苷；鉴伪

# Feasibility Analysis of Anthocyanin Fingerprint Identification of Blueberry Juice

PAN Shao-xiang, YAN Xin-huan\*, TAN Meng-nan, LIU Xue-mei, MENG Xiao-meng, ZHENG Xiao-dong, LI Zhi-cheng, CAO Ning   
(Jinan Fruit Research Institute, All China Federation of Supply & Marketing Co-operatives, Jinan 250014, China)

Abstract: Because of the low output and high cost, the adulteration and counterfeiting of berry juice occur from time to time. A fast and reliable technology for identifying adulteration of berry juice is urgently needed. Anthocyanins in blueberry juice were detected by liquid chromatography. Optimized chromatographic method was that Acclaim $^ \mathrm { T M }$ 120 C18 column ( $2 5 0 \ \mathrm { m m } \times 4 . 6$ mm, $5 ~ { \mu \mathrm { m } }$ ) was used, and the mobile phase was $10 \%$ formate in water, and water:formic acid:acetonitrile (4:1:5); gradient elution was used in the experiment; column temperature was $4 0 \ \textrm {‰}$ ; flow rate was $1 . 0 \ \mathrm { \ m L / m i n }$ ; detection wavelength was $5 3 0 ~ \mathrm { n m }$ . A rapid detection technique for anthocyanin fingerprints in blueberry juice was established. The experiment compared the differences of the fingerprint of anthocyanin in pomegranate juice, raspberry juice and blueberry juice, then discussed the feasibility of the authentic identification of anthocyanin in blueberry juice. The results show that the fingerprint technique of anthocyanin in blueberry juice can be used for the authentic identification of blueberry juice.

Key words: HPLC; blueberry juice; anthocyanin; authentic identification

蓝莓 （Blueberry）为杜鹃花科 （Ericaceae） 越橘属（Vaccinium spp.），多年生落叶或常绿灌木小浆果，原产于北美[1-3]。蓝莓是越橘属植物中营养成分最丰富的种类，除含常规的有机酸、糖以外，还富含维生素 A、E、B 以及多种微量元素，并且富含多种黄酮类、花色苷、多酚类化合物等成分，具有抗氧化活性、抗癌、降低胆固醇、延缓衰老等多种药理活性[4,5]。以蓝莓为代表的浆果类水果有“黄金水果”的美誉，但由于产量低、保鲜难度和成本较高，主要以加工为主，鲜食不足 $5 0 \% ^ { [ 6 ] }$ 。目前，蓝莓多被加工成果汁、浓缩果汁、果酱等。国内果汁行业正以平均每年$2 0 \%$ 的增长率增长，成为食品方面发展最快的产业[7]，浆果汁及其复合饮料市场份额不断增长，形成了新的果汁消费市场。近年来，国内各大果汁饮料厂开始重视浆果果汁的生产，但由于产量低、成本高，浆果果汁掺假、造假、错误标签等导致的食品质量和安全争议不断发生，果汁真伪鉴别技术面临巨大挑战。目前，常用的浆果真伪检测方法，一是基因技术，利用 PCR 技术对果汁中未知物进行鉴伪筛查[8-10]；二是同位素分析法，利用质谱或核磁技术对果汁中 C、H、O 等同位素特征比例精确测定外源添加物含量[11-13]。这些方法在浆果果汁鉴定上有一定的准确性，但过程相对繁琐，不易实现，尤其是测定多品种混合的果汁，方法的准确性、灵敏度和检测范围将会受到很大限制。市售浆果果汁饮品售价高昂且质量参差不齐，消费者又很难通过简单的感官判断加以鉴别，而现行分析检测技术方法无法完全满足上述要求。因此，亟需一种快捷、可靠的浆果果汁掺伪识别技术。

近年来，特征性化学成分的识别是浆果及其果汁检测中兴起的一个热点。通过特征成分检测或建立特征成分指纹图谱来进行果汁鉴伪的研究较多，但由于果汁中化合物天然含量较低和广泛的物种变异性，效果不理想。蓝莓中花色苷种类多，含量高[14,15]，在特征成分指纹图谱鉴伪方面有很大的优势，本研究针对市售蓝莓汁中经常使用树莓汁、石榴汁、葡萄汁等进行以次充好现象，利用液相色谱技术，结合化学计量学统计分析方法，建立了蓝莓汁中花色苷的快速高效指纹图谱鉴伪技术，为蓝莓果汁的质量控制提供一种有效的方法。

# 1 材料和方法

# 1.1 材料与试剂

新鲜的蓝莓、石榴、树莓样品，均购买于济南历下大润发超市。

乙腈为色谱纯，Thermo fisher USA；甲酸为色谱纯，天津市科密欧化学试剂有限公司；实验用水均为超纯水，$1 8 . 2 ~ \mathrm { M } \Omega$ 。

# 1.2 仪器与设备

U3000 反相高效液相色谱仪 （Thermo fisher，USA），包括自动进样器、柱温箱和 DAD 检测器；明澈 D24 UV超纯水仪，美国 Millipore；破壁料理机，九阳 JYL-Y15；KQ-500E 超声仪，昆山市超声仪器有限公司。

# 1.3 供试品制备

为保证果汁的真实性，试验所用蓝莓汁、石榴汁和树莓汁均由实验室自制，将蓝莓、树莓分别放入破壁料理机中打浆，浆体直接压榨取汁。石榴汁由石榴籽粒直接压榨取汁制得。将纯果汁和水 1:2 稀释混匀后，以 $5 0 0 0 ~ \mathrm { r / }$ $\operatorname* { m i n }$ 转速离心 $1 0 ~ \mathrm { m i n }$ ，过 $0 . 4 5 ~ { \mu \mathrm { m } }$ 滤膜，即得供试品溶液。

2.3 基于化学计量学方法的花色苷指纹图谱数据分析

# 2.3.1 花色苷指纹图谱数据化

利用上述方法检测蓝莓、石榴、树莓供试液中的花色苷，每个样品重复一次，记录花色苷的出峰时间和峰面积，结果见表 4（见下页）。将花色苷指纹图谱数据化，便于进一步结合计量学方法进行分析，能够更加直观地反映出不同样品花色苷的差异。

从表 4 中可以看出，蓝莓、石榴、树莓三种样品的花色苷品种和含量各不相同。从蓝莓、石榴、树莓三种样品中共检测出16 种花色苷，为了更加直观地反应三种样品的差异，采用降维分析法中的主成分分析法以及聚类分析法对三种样品做进一步的分析。

# 2.3.2 花色苷指纹数据主成分分析

采用 SPSS 25.0 对不同样品不同保留时间的花色苷峰面积做主成分分析，选取两个主成分，主成分的特征向量和载荷见表5（见下页）。

从表5 中可以看出，第一主成分中贡献率比较大的成 分 是 保 留 时 间 为 9.76、16.383、18.653、19.237、23.693min 处的色谱峰，这 5 种色谱峰在蓝莓汁中出现，未在石榴汁和树莓汁中出现，是蓝莓汁的特征花色苷峰，这说明第一主成分主要反映的是蓝莓汁花色苷的变异信息。石榴汁和树莓汁花色苷色谱峰在第二主成分有较高的响应，说明这一主成分主要反映的是石榴汁和树莓汁花色苷的变异信息。以第一主成分值为横坐标，第二主成分值为纵坐标做散点图（见第 11 页图 3）。第一、第二主成分综合贡献率达到 $9 5 \%$ ，说明前两个主成分基本上可以解释大部分花色苷含量的变异信息。从主成分分析图中可以看出，不同样品在主成分分析图中能够明显的分开，且根据品种划分为三个区域分布在不同的象限中，说明三个品种的果汁中花色苷种类和含量存在显著的差异。

# 3 讨论

# 3.1 蓝莓汁花色苷提取方法的选择

为提高花色苷的提取率，一般采用有机试剂乙醇、甲醇等对样品进行提取[14,16]，本实验研究目的在于鉴伪，无需考虑样品中花色苷具体含量的高低，因此选择将样品直接稀释过滤上机，这样可以避免样品在提取过程中造成的差异，同时避免提取物质杂峰对鉴伪花色苷成分的干扰。

# 3.2 蓝莓花色苷指纹图谱检测方法的建立

本试验建立的蓝莓花色苷指纹图谱检测方法稳定性、重复性和精密度良好，能够检测出蓝莓汁中特征花色苷成分，并且分离度良好，同时对石榴汁和树莓汁样品进行了检测，也能够检出并分离开石榴汁和树莓汁样品中的花色苷成分，这充分说明了蓝莓花色苷指纹图谱检测方法的有效性。

# 3.3 不同果汁指纹图谱色谱峰比对

对蓝莓汁、石榴汁和树莓汁三种果汁的花色苷指纹谱图和数据进行了比对，并利用分析软件进行了主成分分析和聚类分析，聚类分析结果与主成分分析结果相符，结果表明三种果汁的花色苷指纹图谱具有较大的差异性，能够得到明显的区分，这说明不同样品花色苷真实性评价是可行的，花色苷指纹图谱可以作为蓝莓汁真伪鉴别的依据。

# 3 小结

试验结果表明，以臭氧水为基础组配的土壤处理防控山药根结线虫，对山药根结线虫的防效均达到 $8 5 \%$ 以上，取得了很好的防治效果。能够有效控制山药根结线虫的危害，未对山药幼苗产生药害，且对山药产量的恢复或增加效果明显，其中臭氧水 $+$ 生物酵素有机肥 $+$ 噻唑膦处理，防治效果最好，臭氧水 $^ +$ 阿维菌素 $^ +$ 生物酵素有机肥处理，防治效果次之，臭氧水 $^ +$ 生物酵素有机肥处理，最为环保、经济，可以进行大面积推广。

受土壤地形地势以及山药生长习性等一些因素的限制，臭氧水对土壤的处理可能不全面、彻底。采用噻唑膦、阿维菌素等药剂对土壤进行二次处理，可在山药的后续生长期间保护根系，减少根结线虫的侵害机率。

臭氧水对土壤处理过程中，有益微生物也被杀灭，这就需要增加有益菌群。通过增施生物酵素有机肥等生物菌肥，可使土壤中的有益菌群增多，活化土壤，显著改善蔬菜生长状态，提高植株抗根结线虫能力，还可在一定程度上增加土壤中碱解氮、速效磷和速效钾等的含量，减少化肥施用量。
